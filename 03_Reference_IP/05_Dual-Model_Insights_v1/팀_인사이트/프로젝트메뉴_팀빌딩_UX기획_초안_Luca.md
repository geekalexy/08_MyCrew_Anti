# 프로젝트 메뉴 UX 기획안: "에러를 팀 빌딩으로 승화하는 칸반"

> **작성자**: Luca (CTO / System Architect)
> **최종 갱신일**: 2026-04-17
> **도메인**: MyCrew 대시보드 - 프로젝트/팀 메뉴 개편
> **핵심 철학**: AI의 실수를 유저의 '불만/오류'로 취급하지 않고, 신규 입사자와 합을 맞춰가는 '팀 빌딩 프로세스'의 재미로 전환한다.

---

## 1. 개요: 왜 '팀 빌딩' 프로젝트가 필요한가?

현재 사용자는 AI가 지시를 어기면 "봇이 고장 났다"고 느끼고 스트레스를 받습니다. 
이를 해결하기 위해, MyCrew 워크스페이스가 생성되면 **`[프로젝트 0번: 핵심 크루 팀 빌딩]`** 이 기본 디폴트 프로젝트로 세팅되어 안내됩니다. 유저는 칸반 보드를 이용해 AI 직원들의 규칙(Do's & Don'ts)을 카드 형태로 적어나가며, 팀 문화를 설계하는 게임 같은 경험을 하게 됩니다.

---

## 2. 핵심 UI/UX 플로우 (Feedback-to-Rule)

### 💥 Step 1: 에러 타겟팅 (업무 중 발생)
* **상황**: 유저가 마케팅 포스팅 결과물을 보던 중, AI가 "요요~ 소시안입니다! 😘" 라며 너무 가벼운 톤을 쓴 것을 발견함.
* **UX 조치**: 텍스트를 드래그하거나 결과물 하단의 **[🛠️ 룰(Rule)로 승화하기]** 버튼을 클릭함.

### 📝 Step 2: 팀 빌딩 카드 자동 생성 
* 버튼을 누르면 팝업(모달)이 뜨면서 "어떤 규칙을 가르치시겠습니까?" 라고 묻습니다.
* 유저가 *"공식 포스팅에 과도한 이모지나 가벼운 반말 톤앤매너 금지"* 라고 적어 넣습니다.
* 이 내용은 곧바로 **`[프로젝트 0번: 핵심 크루 팀 빌딩]`** 내의 칸반 보드로 전송됩니다.

### 📋 Step 3: 칸반 보드 기반의 팀 룰 진화 프로세스
유저는 시간 날 때 팀 빌딩 칸반 보드에 들어가 룰 카드들의 상태를 옮기며 팀을 육성합니다.

1. `[Backlog / 생각나는 룰]` : 방금 던져둔 가벼운 메모들. ("인사말은 항상 정중하게 할 것")
2. `[To-Do / 변경할 룰]` : "아, 이 룰은 이번 주부터 적용해야겠다" 하고 유저가 드래그 앤 드롭으로 이동시킴.
3. `[In Progress / 동기화 중]` : 카드를 이쪽으로 옮기면, **CMO 에이전트(시스템)**가 작동하며 해당 룰을 AI 크루 전원의 뇌(Context)에 새롭게 적용하기 위해 컴파일(Compile)을 시작함. (진행 중 애니메이션 표시)
4. `[Done / 확정된 그라운드 룰]` : 동기화가 완료되면 카드가 ডান(Done)으로 넘어가고, 백엔드 서버의 `팀_그라운드룰.md` 공식 조항으로 영구 등재됨.

---

## 3. 대시보드 화면 구성 세부안 (Dual View)

### A. 일반 업무 프로젝트 View
* 일반 업무(마케팅, 콘텐츠 작성) 칸반 보드는 태스크의 진행도 스위치 역할 위주로 담백하게 돌아갑니다.
* 에이전트들의 작업 산출물 우측에는 항상 **[📌 팀 빌딩 칸반으로 보내기]** 압정 아이콘이 존재합니다.

### B. "팀 빌딩 / 문화" 전용 대시보드 View
일반 프로젝트와 완전히 분리된 성격의 탭입니다.
* **좌측 패널**: 위에서 설명한 `[Backlog] ➝ [To-Do] ➝ [In Progress] ➝ [Done]` 워크플로우를 갖춘 칸반 보드.
* **우측 패널 (룰 북 뷰어)**: 칸반 보드에서 완료(Done)된 카드들이 자동으로 문서화되어 텍스트로 예쁘게 정렬된 **`[📖 소시안 크루 그라운드 룰 백서]`** 실시간 마크다운 뷰 페이지.

---

## 4. 기대 효과 (연구 목표와 완벽한 정렬)

1. **UX 심리 전환**: "AI가 내 말을 안 듣는다(에러)" ➝ "아직 우리 팀 문화가 덜 학습되었으니 칸반 카드로 하나 가르쳐주마(육성)" 로 유저의 부정적 감정을 긍정적 재미로 바꿔줍니다.
2. **CKS 프레임워크와의 결합**: B팀(실험군)의 요체인 **[회고 및 지식 동기화 지수(KSI)]** 데이터를 뽑아내는 완벽한 GUI 프론트엔드 장치로 작동합니다.
3. **가시성**: 유저가 우측 패널의 `룰 백서`가 점점 촘촘하게 채워지는 것을 보며, "우리 AI 팀이 이만큼 커졌구나" 하는 깊은 락인(Lock-in) 효과를 경험합니다.
