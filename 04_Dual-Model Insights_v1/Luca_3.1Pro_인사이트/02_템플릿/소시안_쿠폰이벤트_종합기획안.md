# 🚀 [기획안] 소시안 Plan C & 무료 쿠폰 프로모션 캠페인

## 1. 📋 캠페인 오버뷰
- **핵심 소재**: 소시안 최상위 모델 'Plan C' 및 할인/기간 무료 체험 쿠폰
- **기획 종류**: (1) 인스타그램 쿠폰 자동발송 이벤트 (2) 블로그 리뷰 이벤트
- **자동화 액션**: 인스타그램 게시물에 특정 댓글 작성 시 ➡ 소시안 자동 DM 기능으로 쿠폰 코드 즉각 전송

## 2. 📝 채널별 콘텐츠 템플릿

### [A] 인스타그램 릴스 & 캐러셀 (메인 쿠폰 이벤트)
* **목적**: 인게이지먼트(댓글) 폭발 유도 및 유저에게 '소시안 DM 자동화'의 편리성 직접 체감시킴
* **후킹 텍스트**: "아직도 수동으로 답장하세요? 이 릴스에 댓글 달면 DM으로 쿠폰 드려요. (소시안 Plan C)"
* **본문 템플릿**:
  ```text
  🎉 [소시안 최고 등급 Plan C, 한정판 무료 체험 코드를 쏩니다!]
  
  인스타 자동화, 막상 세팅하려니 매니챗은 너무 복잡하셨죠?
  국내 최적화! '매니챗보다 쉬운' 소시안의 최상위 Plan C를 경험해보세요.
  
  👇 **참여 방법 (너무 쉬움 주의)** 👇
  1. 본 게시글에 "최고" 라고 2글자만 댓글로 달아주세요!
  2. 여러분의 DM으로 [무료 체험 쿠폰 코드]를 쏴드립니다! ✉️
  
  여러분의 계정에도 자동화를 달고 싶다면, 지금 바로 체험해보세요!
  ```

### [B] 공식 홈페이지 & 블로그 (리뷰 이벤트 공지)
* **목적**: 진성 유저들의 장문 후기 생성을 통해 정보성 검색 SEO 확보 및 신뢰도 구축
* **본문 템플릿**:
  ```text
  [이벤트] 소시안 Plan C 써보신 분? 솔직한 리뷰 남기고 할인받자! 🎁
  
  수많은 공구 셀러분들이 이미 소시안의 매력에 빠져 있습니다.
  여러분의 블로그에 솔직담백한 '소시안 사용 후기'를 남겨주세요!
  
  📌 **참여 방법**
  1. 본인 블로그에 소시안 연동 과정 및 사용 후기 작성 (최소 사진 3장)
  2. 필수 해시태그: #소시안 #인스타DM자동화 #공구셀러필수템 #매니챗대안
  3. 본 공지글의 비밀댓글로 '내 블로그 리뷰 링크' 남기기
  
  🎁 **참여 혜택**: 베스트 리뷰어 10분께 [Plan C 6000원 할인권] 증정! 
  기한: 202X년 X월 X일까지
  ```

### [C] 유튜브 쇼츠 (기능 체감형 바이럴)
* **목적**: 숏폼 알고리즘을 통한 타겟 도달 및 유입 (인스타 쿠폰 이벤트로 토스)
* **자막/스크립트 템플릿**:
  ```text
  (시각: 새벽 2시, 쌓여가는 인스타 DM에 괴로워하는 셀러 모습)
  🗣️ "공구 대박나서 댓글 1000개 달렸는데 언제 다 답장하고 잘래?"
  (시각 전환: 소시안 세팅 후 자동으로 DM이 빠르게 날아가는 사이다 화면)
  🗣️ "소시안 Plan C! 세팅 한 번이면 자는 동안에도 판매가 치솟는다!"
  👉 고정 댓글의 인스타 링크 누르고, 댓글 달아서 한 달 무료 쿠폰부터 받아가세요!
  ```

## 4. 🤖 캠페인 수행 워크플로우 & 크루의 역할 및 참여 순서

이 캠페인이 가장 높은 ROAS(스노우볼)를 굴릴 수 있도록, 각 크루(에이전트)들이 다음과 같은 워크플로우로 유기적 협업을 진행합니다.

1. **[전략기획 크루] - Campaign Architecture Set-up**
   - **역할**: 캠페인의 큰 그림(예산, 쿠폰 발급 수, 기간)을 대표님과 조율하여 픽스.
   - **순서**: 가장 먼저 투입되어 이벤트 전체 로드맵 승인.

2. **[카피라이팅 크루] - 텍스트 엔진 가동**
   - **역할**: 인스타 캐러셀 대본, 블로그 SEO 공지글, 릴스 자막 최적화. 
   - **핵심 미션**: 공구 셀러들의 페인포인트를 자극하여 유저가 홀린 듯 "최고"라는 댓글을 달 수밖에 없는 후킹 문구 생성.

3. **[디자인/영상 크루] - Visualizing**
   - **역할**: 카피 크루의 글을 받아 'Plan C'의 프리미엄을 상징하는 비주얼 에셋(Socian Blue + 고급화된 UI 요소)을 생성.
   - **순서**: 텍스트 완성 직후 썸네일, 숏폼 배경 에셋을 렌더링.

4. **[퍼포먼스/테크 크루 (아리)] - Automation Set-up (가장 중요!)**
   - **역할**: 위 1~3번이 재료라면 이건 심장입니다. 실제 소시안 백엔드(혹은 인스타 API)에 **"소시안최고" 키워드를 조건으로 거는 DM 자동화 플로우를 완벽하게 세팅하고 테스트 전송**을 수행합니다.
   - **순서**: 오픈 전일(D-1) 자정까지 세팅 및 자체 교차 검증 완료.

5. **[어낼리틱스(데이터) 크루] - Feedback Loop**
   - **역할**: D+3, D+7 시점에 "댓글 전환율 대비 실제 쿠폰 등록 수", "블로그 리뷰 생성 목표치" 등을 크롤링/파싱하여 성과를 `주간보고서_템플릿`에 맞춰 대표님께 브리핑.
