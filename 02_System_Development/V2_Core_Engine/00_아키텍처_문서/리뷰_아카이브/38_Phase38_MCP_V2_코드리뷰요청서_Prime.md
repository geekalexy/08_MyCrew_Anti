## 3. 스모크 테스트 체크리스트

코드 리뷰와 함께 아래 항목을 검증해주세요:

| # | 테스트 항목 | 확인 방법 | 통과 기준 |
|---|-----------|----------|----------|
| 1 | **MCP 서버 기동** | `node mcp-server.js` 실행 | 에러 없이 부팅, 클라이언트 연결 수락 가능 |
| 2 | **Resource 읽기** | MCP 클라이언트에서 칸반 리소스 요청 | JSON 형태로 태스크 리스트 정상 반환 |

| 4 | **대시보드 정상 동작** | 브라우저에서 기존 대시보드 접속 | 칸반 보드 렌더링, 카드 이동, 로그 드로어 정상 |
