### ② KSI를 두 가지로 분리해야 한다

루카와 소넷, 서로 다른 KSI를 정의했다. 충돌이 아니라 — 둘 다 필요하다.

| | 정의 | 측정 방법 |
|---|------|---------|
| **루카 KSI-R** | 금지 조항이 다음 스프린트에 반영된 비율 (룰 생존율) | 규칙 이행 체크리스트 |
| **소넷 KSI-S** | 교차 평가 후 상대 개념 흡수율 (의미론적 이전율) | 코사인 유사도 델타 |

```
KSI-R (Rule Compliance):  루카 정의 → MyCrew 실무 측정
KSI-S (Semantic Transfer): 소넷 정의 → 학술 논문 핵심 지표
```

### ⑦ KSI 베이스라인 차감 — 소넷 단독 기여

같은 프롬프트를 받으면 원래부터 유사한 출력이 나온다.
차감 없이는 "교차 평가 효과"와 "주제 유사성"이 구분 불가.

```python
# KSI-S 수정 공식 (베이스라인 차감 필수):
KSI_delta_A = CS(A_post, B_pre) - CS(A_pre, B_pre)
KSI_delta_B = CS(B_post, A_pre) - CS(B_pre, A_pre)
KSI_S = (KSI_delta_A + KSI_delta_B) / 2

# 심사위원 예상 반박 차단:
# "원래 같은 주제라서 비슷한 거 아니냐" → 베이스라인 차감이 증거
```
