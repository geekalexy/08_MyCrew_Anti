## 4. 핵심 인사이트 목록

1. **루카 = 건축가, 소넷 = 인테리어 디렉터** (프라임 정의)
2. **"기획자만 있으면 온보딩 1일차 이탈, 아키텍트만 있으면 유저가 귀찮아서 묵사발"** (루카)
3. **Data Input(소넷) vs Data Usage(루카) = 동전의 양면** (루카)
4. **문서 없는 영역이 진짜 차이점** — 만들지 않은 것이 특성을 드러냄 (프라임 Finding 3)
5. **"듀얼 크루는 사치가 아니라 보험"** (프라임 최종 한 줄)
