# 분양 데모 시나리오

## 목적
지인에게 AI 에이전트 회사가 어떻게 작동하는지 **5분 안에** 보여주기

---

## 사전 준비 (데모 전)

1. 지인 회사를 Paperclip에 Add company로 미리 생성
2. 에이전트 최소 구성: CEO + 1~2명 (지인 업종에 맞게)
3. 샘플 이슈 2~3개 미리 생성
4. 텔레그램 봇 생성 + 지인 폰에 봇 추가

---

## 데모 흐름 (5분)

### 1단계: 첫 연결 (30초)

```
[지인 폰에서]
텔레그램 → 봇 검색 → /start

봇 응답:
"연결 완료! 명령어 안내..."
```

**포인트**: "이것만 하면 됩니다. 설치 끝."

---

### 2단계: 회사 현황 확인 (1분)

```
지인: "지금 뭐하고 있어?"

봇 응답:
"현재 에이전트 현황입니다.
- CEO: 대기 중
- CMO: 마케팅 전략 수립 중
..."
```

```
지인: /iss

봇 응답:
[SOC-1 마케팅 전략 수립]
[SOC-2 콘텐츠 기획]
[SOC-3 ...]
```

**포인트**: "텔레그램 하나로 회사 전체를 파악할 수 있습니다."

---

### 3단계: 에이전트에게 일 시키기 (1분)

```
지인: /newiss

봇: 담당자를 선택해주세요
    [CEO] [CMO] [Content Marketer]

지인: [CMO] 클릭

봇: 프로젝트를 선택해주세요
    [Onboarding] [마케팅]

지인: [마케팅] 클릭

봇: 이슈 제목을 입력해주세요

지인: "인스타그램 마케팅 전략 기획해줘"

봇: 이슈 생성 완료!
```

**포인트**: "이렇게 지시하면 AI 직원이 알아서 일합니다."

---

### 4단계: 에이전트 작업 결과 확인 (1분)

```
[미리 만들어둔 샘플 이슈 클릭]
지인: /iss SOC-1

봇:
"SOC-1 마케팅 전략 수립

CEO가 작업 완료:
- 타겟 고객 분석 완료
- 채널별 전략 수립
- 예산 배분안 작성
..."
```

**포인트**: "이렇게 보고가 올라옵니다. 확인하고 승인하면 됩니다."

---

### 5단계: 승인 처리 (30초)

```
봇: "대표님, SOC-1 건에 대해 승인 요청이 들어왔어요~
     [승인]"

지인: [승인] 클릭

봇: "SOC-1 승인 완료!"
```

**포인트**: "출퇴근 중에도 이렇게 승인하면 AI가 다음 단계를 진행합니다."

---

### 6단계: Paperclip 대시보드 (1분)

```
[지인 PC 또는 모바일에서 대시보드 접속]

"여기서 더 자세하게 볼 수 있습니다."
- 에이전트가 일하는 모습 (실시간)
- 이슈 상세 내용
- 직접 설정 변경, 에이전트 추가
```

**포인트**: "텔레그램은 간편 리모컨, 대시보드는 풀 컨트롤러입니다."

---

## 마무리 멘트

> "지금 보신 것처럼, AI 직원들이 24시간 일합니다.
> 당신은 텔레그램으로 지시하고 승인만 하면 됩니다.
> 한 달 무료로 써보시고 괜찮으시면 계속 쓰시면 됩니다."
