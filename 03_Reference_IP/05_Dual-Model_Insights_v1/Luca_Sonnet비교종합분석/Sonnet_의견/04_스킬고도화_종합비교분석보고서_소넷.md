## 3. C레벨 스킬 비교

### 3-1. 루카의 C레벨 직책 분리 (CPO·CQA·CHRO)

| 직책 | 역할 | 전용 스킬 |
|------|------|---------|
| **CPO** (최고 프로덕트 책임자) | 태스크 병렬 할당·스케줄 감리 | `/태스크_병렬할당`, `/스케줄_감리` |
| **CQA** (최고 품질 통제관) | Red Teaming 1차 비평 | `/1차_실무비평` (Reject↔Approve) |
| **CHRO** (최고 인사감사 책임자) | 컴플라이언스·스킬감사·채용 | `/크루_컴플라이언스_정기평가`, `/스킬_미아_추적`, `/스킬_큐레이션_보고`, `/채용_수요분석` |

**루카만의 강점**: CHRO 직책의 **`/스킬_미아_추적`** — "크루가 뇌피셜(환각)로 일했는지 시스템 로그 추적"
→ 환각(Hallucination) 방지를 **스킬 단위**로 설계한 점이 독보적

### 3-2. 루카의 칸반 흐름 (4단계)

```
[To-Do] → 아리 접수·프롬프트 변환 → CMO 이관
[In Progress] → CMO가 크루에 병렬 지시
[CMO Review] → 실무 크루 1차 제출 → CMO 심사 (반복 가능)
[CEO Review] → 완벽 통과한 단 하나의 마스터피스만
```

→ **CEO Review 열 = 마스터피스만 올라오는 구조** — 대표 피로도 제로화 설계

### 3-3. 소넷의 C레벨 스킬 (Category 0, 13개)

| 강점 | 내용 |
|------|------|
| 번호 체계 | 0-1~0-13 명확한 참조 가능 |
| 이행 감독 | 0-13: 일간/주간 이행률 집계 + 개선 사이클 |
| Phase 설계 | Phase1(CMO겸직)→Phase2(PM분리) 로드맵 |
| 문서 완성도 | 입력·출력·상세 전체 명시 |
