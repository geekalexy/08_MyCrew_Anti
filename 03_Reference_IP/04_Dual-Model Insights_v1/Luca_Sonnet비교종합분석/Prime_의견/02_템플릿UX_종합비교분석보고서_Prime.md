## 2. 카피 품질·후킹 비교

### 🔥 Luca — "한 방 기획안" 스타일
Luca의 `소시안_쿠폰이벤트_종합기획안.md`는 하나의 파일에 캠페인 전체를 담은 **"한 방 기획서"** 성격. 특히 카피의 감성적 후킹이 강렬함:

> _"아직도 수동으로 답장하세요? 이 릴스에 댓글 달면 1초 만에 DM 쏴드립니다."_

> _"새벽 2시, 쌓여가는 인스타 DM에 괴로워하는 셀러 모습"_

- **카피라이팅 전문성**: 감정 자극(Before/After), 긴급성, 셀러의 페인포인트를 정확히 타격하는 후킹 텍스트
- **유튜브 쇼츠 시나리오**: 영화식 시각 전환(새벽 2시 → 사이다 화면)까지 기획
- **단점**: 모든 것이 한 파일에 뭉쳐 있어 개별 채널 분리 어려움

### 📐 Sonnet — "모듈형 매뉴얼" 스타일
Sonnet의 템플릿은 채널별로 엄격히 파일을 분리하고, 각 채널 내에서도 다수의 서브템플릿을 제공:

> _"지금 이 게시물에 '쿠폰' 댓글을 달면 소시안이 자동으로 DM을 보내드릴게요 📩"_

- **구조적 완성도**: 캐러셀은 슬라이드별(1~4장) 비주얼 설명 + 텍스트 카피가 분리, 릴스는 `[장면N — 시간: 역할]` 포맷
- **변수 시스템**: `{{N}}`, `{{금액}}`, `{{이벤트 기간}}` 등 5종 변수 목록 테이블로 정리
- **기존 자산 참조**: `> 참고 기존 자산: 06_소시안자료/컨텐츠/인스타게시물/` 직접 링크
- **단점**: 카피가 정돈되었으나 Luca 대비 감정 자극이 약하고 "교과서적"

## 3. 크루 역할(R&R) 및 워크플로우 비교

### Luca
5개 크루 역할을 **실행 순서대로** 나열, 각각의 핵심 미션을 한 줄로 정의:
1. 전략기획 크루 → "캠페인 로드맵 승인"
2. 카피라이팅 크루 → "홀린 듯 댓글 달 수밖에 없는 후킹 문구"
3. 디자인/영상 크루 → "에셋 렌더링"
4. 퍼포먼스/테크 크루(아리) → **"가장 중요! DM 자동화 플로우 세팅"**
5. 어낼리틱스 크루 → "성과 브리핑"

**특징**: "아리의 자동화 세팅이 심장"이라는 **강한 우선순위 표현**으로 실무 중요도 직관 전달

### Sonnet
7단계 Phase 워크플로우를 **Mermaid-like 순차 다이어그램**으로 표현:
```
Phase 1: 전략 수립 → Phase 2: 콘텐츠 제작 (병렬) → Phase 3: SEO →
Phase 4: 검토/승인 → Phase 5: 발행 → Phase 6: 모니터링 → Phase 7: 결과 보고
```

**특징**: 
- "Phase 2 병렬 진행" 같은 프로세스 설계 관점
- KPI 지표 테이블 별도 제공 (댓글 수, DM 발송 수, 리뷰 제출 수, 전환율)
- 발행 시간대 테이블 (인스타: 오전 7~9시 / 블로그: 오전 10~11시)

## 4. 범용 vs 전용 템플릿

### Luca — 범용 프레임 포함
`인플루언서_캠페인_기획_템플릿.md`와 `주간_보고서_템플릿.md`는 소시안에 한정되지 않는 **범용 양식**:
- 어떤 고객사에도 재사용 가능한 "빈 프레임"
- MyCrew의 SaaS 재사용성에 기여
- 다만 내용이 거의 없어 단독으로는 가치 제한적

### Sonnet — 소시안 특화 밀착
모든 파일이 소시안 Plan C/쿠폰이벤트에 100% 밀착:
- 소시안 공식 URL(`socian.alocados.io`), 인스타 계정 직접 삽입
- 이미지 자산 경로 직접 참조
- 범용성은 떨어지지만 **당장 실행 가능한 정도**가 높음

## 🏆 최종 비교 결론 — 템플릿 UX

### Sonnet이 우세한 영역
1. **볼륨과 체계**: 3.7배의 콘텐츠 양, 채널별 파일 분리, 슬라이드/장면 단위 구성
2. **변수 시스템**: 미확정 값을 `{{변수}}` 처리하여 실무 적용 시 즉시 치환 가능
3. **기존 자산 연동**: 소시안 기존 이미지/영상 폴더를 직접 참조
4. **운영 체계 분리**: 스케줄, 워크플로우, KPI를 독립 파일로 관리

### Luca가 우세한 영역
1. **카피 감성 품질**: 감정 자극·Before/After·긴급성 활용이 실제 퍼포먼스 마케터 수준
2. **한 방 기획서 완결성**: 하나의 파일로 즉시 실행 브리핑 가능
3. **범용 프레임 제공**: SaaS 비즈니스 확장 시 어떤 고객사에도 재사용 가능
4. **우선순위 직관성**: "아리가 심장"이라는 직관적 표현으로 팀 포커스 정렬

### Prime 권장: 최적 병합 전략
> **Sonnet의 모듈형 구조 + Luca의 카피 감성 = 최종 템플릿 라이브러리**

1. 파일 구조 → Sonnet의 채널별 분리 채택
2. 카피 텍스트 → Luca의 후킹 문구·감성 워딩을 Sonnet 슬라이드 구조에 삽입
3. 범용 프레임 → Luca의 `인플루언서_캠페인_기획_템플릿.md`를 MyCrew 표준 프레임으로 승격
4. 변수 시스템 → Sonnet의 `{{변수}}` 체계를 전사 표준으로 채택
