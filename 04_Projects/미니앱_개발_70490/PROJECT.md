# PROJECT: 미니앱 개발

## 🎯 프로젝트 개요
MyCrew의 핵심 기능들을 원격에서 제어할 수 있는 텔레그램 미니앱을 개발합니다. 이 프로젝트는 사용자 데이터의 엄격한 격리(strict_isolation)를 보장하며, 안정적이고 직관적인 컨트롤 인터페이스를 제공하는 것을 최우선 목표로 합니다.

## 🛑 대원칙 및 톤앤매너
- **Strict Isolation First:** 모든 아키텍처 설계와 데이터 처리 과정에서 '데이터 격리'를 최우선 원칙으로 삼습니다.
- **Telegram Native Experience:** 텔레그램 사용자에게 가장 익숙하고 매끄러운 UI/UX를 제공하여 학습 비용을 최소화합니다.
- **MVP Focus:** 가장 핵심적인 원격 제어 기능에 집중하여 첫 스프린트를 진행하고, 이후 빠르게 기능을 확장해 나갑니다.
- **Documentation-Driven:** 모든 개발은 명확하게 작성된 기술 문서(PRD, API 명세)를 기반으로 진행하여 커뮤니케이션 오류를 방지합니다.

## 🚀 성공 지표
- 첫 스프린트 내 MVP(최소 기능 제품) 버전의 내부 테스트 릴리즈 완료.
- 'strict_isolation' 요건을 충족하는 데이터 모델 및 아키텍처 설계에 대한 동료 리뷰 통과.
- 핵심 원격 제어 기능의 평균 응답 시간 500ms 미만 달성.
