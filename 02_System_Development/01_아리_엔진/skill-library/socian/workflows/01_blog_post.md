# 소시안 블로그 포스트 워크플로우

## 🎯 목표
소시안 브랜드 블로그 포스트를 SEO 최적화 + 전환 유도 구조로 자동 생성한다.

---

## 📋 실행 단계 (Step-by-Step)

### Step 1: 키워드 & 주제 리서치 (OLLIE)
```
담당: OLLIE (데이터 분석)
입력: 대표님 지시 또는 칸반 카드 제목
출력:
  - 타겟 메인 키워드 1개
  - LSI 키워드 3~5개
  - 예상 검색량 범위 (낮음/중간/높음)
  - 경쟁 강도 (낮음/중간/높음)
  - 추천 포스트 각도(Angle) 2가지
```

### Step 2: 초안 생성 (NOVA or PICO)
```
담당: NOVA (SEO 기반 논리적 구조) 또는 PICO (감성 Hook 기반)
CCB 참조: Step 1 키워드 리서치 결과
출력 구조:
  [제목] 메인 키워드 포함 40자 이내
  [서론] 독자 통증 자극 3줄 + 소시안 해결책 1줄
  [본론 H2×3] 각 소제목 아래 300~500자
  [소시안 기능 연결] 본론 중 최소 2곳 자연스러운 녹여넣기
  [CTA] 무료 체험 유도 링크 + 해시태그 5개
```

### Step 3: 검수 & 포맷팅 (LUMI)
```
담당: LUMI (디자인 포맷팅)
체크리스트:
  □ 제목에 메인 키워드 포함 여부
  □ 본문 1000자 이상 여부
  □ 소시안 브랜드 톤 준수 여부
  □ CTA 문구 명확성
  □ 네이버 블로그 발행 포맷 호환 여부
출력: 발행 준비 완료 마크다운 파일
```

### Step 4: 대표님 승인 → Done
```
ARI가 최종 산출물을 대표님께 보고
수정 있을 시: Step 2 복귀 (PICO 재작성)
승인 시: 칸반 Done 이동 + Self-Learning 로그 기록
```

---

## 📐 품질 기준 (Done Criteria)
- 네이버/구글 키워드 검색 결과 1~3페이지 노출 가능 수준
- 소시안 무료 체험 CTR 목표: 포스트 조회수의 3% 이상
- 대표님 수정 횟수: 0~1회
