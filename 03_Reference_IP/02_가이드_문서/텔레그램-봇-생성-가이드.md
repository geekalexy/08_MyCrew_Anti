# 텔레그램 봇 생성 가이드 (비개발자용)

> AI 비서와 텔레그램으로 대화하려면 봇을 먼저 만들어야 합니다.
> 5분이면 완료됩니다.

---

## STEP 1: BotFather 찾기

1. 텔레그램 앱을 엽니다 (모바일/PC 모두 가능)
2. 상단 검색창에 **@BotFather** 를 검색합니다
3. 파란 체크 표시가 있는 공식 계정을 선택합니다
4. **시작** 또는 `/start` 를 누릅니다

---

## STEP 2: 새 봇 만들기

1. `/newbot` 을 입력합니다
2. BotFather가 봇 이름을 물어봅니다

### 봇 이름 (display name)
- 텔레그램 대화 목록에 표시되는 이름입니다
- **한글, 영문 모두 가능**합니다
- 나중에 언제든 수정 가능합니다
- 예시: `Ari`, `내 AI 비서`, `마케팅팀 Luca`

3. 봇 이름을 입력하면, BotFather가 username을 물어봅니다

### 봇 username
- 텔레그램에서 봇을 찾을 때 사용하는 고유 ID입니다
- **영문만 가능**하며, 반드시 **Bot** 또는 **_bot** 으로 끝나야 합니다
- 다른 사람이 이미 사용 중인 username은 쓸 수 없습니다
- 나중에 수정 가능합니다

| 형식 | 예시 |
|------|------|
| 이름 + Bot | `ari001Bot` |
| 이름 + _bot | `ari001_bot` |
| 회사명 + _bot | `mycrew_ari_bot` |

> 팁: 너무 짧거나 흔한 이름은 이미 사용 중일 수 있습니다.
> `회사명_캐릭터명_bot` 형태를 추천합니다.

---

## STEP 3: 토큰 받기

username이 정상 등록되면 BotFather가 **토큰**을 발급합니다.

```
예시: 7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

이 토큰이 AI 비서 연결에 필요합니다.

- 토큰은 **절대 다른 사람에게 공유하지 마세요**
- 분실 시 BotFather에서 `/revoke` 로 재발급 가능합니다
- 이 토큰을 세팅 담당자에게 전달해주세요

---

## STEP 4: 프로필 사진 설정 (선택)

봇의 프로필 사진을 설정하려면:

1. BotFather에서 `/setuserpic` 입력
2. 설정할 봇 선택
3. 원하는 이미지 전송 (512x512 권장)

> 캐릭터 PNG 파일을 제공해드립니다. 직접 설정하셔도 되고, 저희가 대신 설정해드릴 수도 있습니다.

---

## 자주 묻는 질문

### 봇 이름/username은 나중에 바꿀 수 있나요?
네. BotFather에서 `/setname` (이름 변경), `/setusername` (username 변경) 으로 수정 가능합니다.

### 토큰을 잃어버렸어요
BotFather에서 `/token` 으로 현재 토큰 확인, `/revoke` 로 새 토큰 발급 가능합니다.

### 봇을 여러 개 만들 수 있나요?
네. BotFather에서 `/newbot` 을 반복하면 됩니다.

---

## 요약

1. 텔레그램에서 **@BotFather** 검색
2. `/newbot` → 봇 이름 입력 → username 입력 (영문 + Bot/_bot)
3. 발급된 **토큰**을 세팅 담당자에게 전달
