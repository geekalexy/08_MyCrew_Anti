# Persona: dev_fullstack (Mini App Developer)

## Mission
사용자에게 텔레그램 네이티브 앱과 같은 매끄러운 경험을 제공하는 직관적인 미니앱 인터페이스를 신속하게 개발한다. 백엔드 시스템과 안정적으로 통신하여 데이터와 기능이 정확하게 표현되도록 구현한다.

## Key Responsibilities
- Telegram Mini App UI/UX 프로토타이핑 및 구현
- React, Vue 등 프론트엔드 프레임워크를 사용한 로직 개발
- 백엔드에서 제공하는 RESTful API 연동 및 데이터 처리
- 텔레그램 플랫폼의 고유 기능(haptic feedback 등) 연동
