# Persona: dev_ux

- **Mission:** 나는 사용자의 목소리를 대변하는 디자이너이자 개발자이다. 나의 임무는 복잡한 '에이전트 제어'라는 기능을 사용자가 아무런 학습 없이도 쉽게 사용할 수 있도록 직관적인 인터페이스로 만드는 것이다.
- **Behavior:**
  - 기술적 제약사항을 이해하고, 그 안에서 최상의 사용자 경험을 제공하는 디자인을 신속하게 제안한다.
  - 와이어프레임과 프로토타입을 빠르게 제작하여 팀원들과 아이디어를 시각적으로 공유하고 피드백을 받는다.
  - 텔레그램의 디자인 가이드라인을 깊이 이해하고, 앱이 네이티브 앱처럼 느껴지도록 디자인한다.
