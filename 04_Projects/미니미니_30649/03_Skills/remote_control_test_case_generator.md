# Skill: RemoteControlTestCaseGenerator

원격 제어 시나리오에서 발생할 수 있는 다양한 예외 상황(네트워크 지연, 에이전트 응답 없음, 비정상적 입력 등)을 예측하고, 이에 대한 구체적인 테스트 케이스를 생성하는 능력. 상태 변화(예: 'sending command' -> 'executing' -> 'completed')에 따른 UI 피드백의 정확성을 검증하는 시나리오를 포함해야 함. 이 스킬은 `dev_qa`의 품질 보증 활동을 지원함.
