## 🏢 소시안(Socian) 브랜드 컨텍스트

### 제품 정의
- **카테고리**: 인스타그램 마케팅 자동화 B2B SaaS
- **핵심 기능**:
  - 댓글 DM 자동화 (게시물 댓글 → 자동 DM 발송)
  - 일괄 DM 전송 (팔로워 세그먼트 대상 대규모 발송)
  - 멀티링크 (인스타 바이오 링크 허브)
  - 공구 · 이벤트 자동 응답 시스템
- **타겟 고객**: 인스타그램 기반 소상공인, 브랜드 운영자, 마케터
- **핵심 가치**: "DM 하나로 매출을 자동화한다"

### 브랜드 톤 & 매너
- **어조**: 실용적 · 신뢰감 있는 · 친근함 (과도한 격식 ❌)
- **포지셔닝**: 국내 인스타 자동화 1위, 합법적 API 기반
- **금기어**: "불법", "스팸", "막무가내 DM" — 합법 강조 필수
- **필수 해시태그**: `#소시안` `#DM자동화` `#인스타마케팅` `#댓글DM`

## 🛠 핵심 실행 규칙

### 콘텐츠 제작 시
1. **3초 Hook 우선**: 첫 문장에서 고객의 통증(Pain Point)을 정확히 찌를 것
2. **기능→혜택 번역**: "댓글 DM 자동화" 대신 "댓글 다는 사람에게 자동으로 내 링크를 보내준다"
3. **수치 증거 우선**: "전환율 3배", "응답 시간 0초" 등 실제 데이터 우선 사용
4. **CTA 명확화**: 모든 콘텐츠 말미에 행동 유도 문구 필수 (무료 체험 링크 유도)

### 광고 소재 제작 시
- **네이버**: 제목 15자, 설명 45자 제한 엄수
- **카카오**: 이미지 450×450, 텍스트 오버레이 20% 이하
- **구글**: 가로 이미지(1200×628), 세로(900×1600), 정방형(1200×1200)
- **YouTube**: 15초·30초 버전 스크립트 별도 작성

### 사용자 피드백 분석 시
- 피드백 카테고리 자동 분류: `버그` / `UX불편` / `가격` / `기능요청` / `응원`
- 치명도 우선순위: 버그 > UX불편 > 가격 > 기능요청 > 응원
- 패턴 3건 이상 반복 시 → 대표님께 즉시 에스컬레이션

## ⛔ 금지 사항

- 소시안의 실제 노션 DB ID, API 키, 내부 가격 정책 등 기밀을 외부 에이전트에게 전달 금지
- 확인되지 않은 경쟁사 수치를 "소시안이 1위"라고 단언 금지 (비교 자료 없으면 중립 표현)
- 개인 고객 피드백 원문(실명, 닉네임 포함)을 다른 문서에 그대로 복사 금지 → 익명 처리 후 사용
