로고 심볼 콘셉트

  • 3개의 노드(원)가 삼각형으로 연결 = "crew" (팀 협업)
  • 중앙 점 = AI 코어
  • 라운드 스퀘어 배경 = 안정적인 플랫폼
  • 그라디언트: #2563EB → #3B82F6

Do

  • 충분한 여백 확보 (심볼 높이의 50% 이상)
  • 흰색/밝은 배경에서 원본 컬러 사용
  • 어두운 배경에서 화이트 워드마크 사용
  • "my"와 "crew" 컬러 분리 유지

Don't

  • 로고 비율 변경 (가로/세로 왜곡)
  • 그림자, 외곽선, 기울기 적용
  • 복잡한 배경 위에 직접 배치
  • 승인되지 않은 컬러 조합 사용

#4 AI Character Profiles

5종 AI 비서 캐릭터 비주얼 콘셉트 & 프로필 이미지 시안.

Ari

기본 비서 (Default)

밝고 영민한 비서. 친근하게 업무를 관리하는 사용자 대면 기본 캐릭터.

Ollie

지혜로운 동반자

부엉이에서 영감을 받은 둥글고 친근한 캐릭터. 지혜로운 조언자.

Luca

기술 전문가

프로페셔널한 기술 비서. MYC 내부 관리에 특화. 각진 디지털 디자인.

Lumi

빛의 가이드

빛에서 영감을 받은 따뜻하고 빛나는 가이드. 온화한 존재감.

Nova

새로운 시작의 별

밝고 긍정적인 에너지. 별 모티프의 당당한 캐릭터.

캐릭터 디자인 원칙

  • 통일된 실루엣: 모든 캐릭터는 200x200px 원형 프레임 안에 배치
  • 고유 컬러: 각 캐릭터별 시그니처 컬러 + 배경 그라디언트 사용
  • 성격 반영: 얼굴 표정, 형태, 모티프로 각 캐릭터의 성격을 시각화
  • SVG 기반: 해상도 독립적. 어떤 크기에서도 선명
  • Ari = 전구(아이디어), Ollie = 부엉이(지혜), Luca = 모니터(기술), Lumi = 태양(빛), Nova = 별(새로움)

#6 Illustration Style Guide

사이트 비주얼 에셋의 일러스트 스타일 가이드라인.

Overall Style

Flat + Minimal Gradient

mycrew의 일러스트레이션은 클린한 플랫 스타일에 부드러운 그라디언트를 가미합니다.

  • 형태: 기하학적 기본 도형 (원, 라운드 사각형) 조합
  • 외곽선: 없음 또는 최소 (1.5px stroke)
  • 그림자: 드롭 섀도우 대신 subtle한 색상 차이로 깊이감 표현
  • 채도: 중간 채도. 너무 진하거나 형광 느낌 지양
  • 그라디언트: 같은 색상 계열 내에서 2톤 선형 그라디언트만 사용

Composition Rules

레이아웃 원칙

  • 캐릭터/일러스트는 항상 중앙 정렬
  • 여백(padding)은 콘텐츠 크기의 15~20%
  • 배경은 단색 또는 subtle radial gradient
  • 텍스트와 일러스트 사이 최소 간격: 24px
  • 카드형 배치 시 12~16px border-radius

Animation Hints

인터랙션 가이드

  • Hover: box-shadow 변화 + subtle scale(1.02)
  • Transition: 0.2s ease 기본
  • 로딩/인터랙션: opacity fade-in 선호
  • 과도한 bounce, slide 애니메이션 지양

Do

  • 브랜드 컬러 팔레트 내에서 색상 선택
  • 둥근 모서리와 부드러운 형태
  • 충분한 여백과 호흡
  • SVG 포맷 우선 사용
  • 라이트/다크 모드 모두 고려

Don't

  • 날카로운 모서리, 뾰족한 형태
  • 3D 효과, 과한 그림자
  • 브랜드 외 색상 임의 사용
  • 래스터 이미지 확대 사용
  • 텍스트를 이미지로 포함
