## 2. 📊 시스템이 사용하는 API 전체 현황

| API | 사용 위치 | 현재 무료 제한 |
|---|---|---|
| **Gemini 2.5 Flash** | Image Lab 분석 (`analyzeImage`), Video Lab 생성 | **10 RPM / 250 RPD** |
| **Imagen 3** (`imagen-3.0-generate-001`) | Image Lab 이미지 생성 (1차 시도) | 별도 쿼터, 한국 리전 제한 있음 |

> Pro 모델에 요청 시 100% 429 에러 리턴.

## 4. 🌱 Pollinations 티어 로드맵

| 티어 | Pollen/hour | 조건 | 코드에서 사용할 모델 |
|---|---|---|---|
| 🍄 **Spore** (현재) | 0.01 | 계정 인증 | `flux-schnell` ← 현재 |
| 🌱 **Seed** | 0.15 | 7+ dev points (매주 자동 평가) | `flux` |
| 🌸 **Flower** | 0.4 | 앱 퍼블리시 + Seed 선행 | `flux` + 고급 모델 |

**Seed dev points 구성** (7점 필요):
- Account age: 0.5pt/month (max 6)
- Public commits (90일): 0.1pt each (max 3)
- Original repos (public, non-empty): 0.5pt each (max 1)
- Stars on original repos: 0.1pt each (max 5)

## 5. 📐 Auto Loop 설계 현황 (Image Lab)

Full Auto Loop 쿨다운: **45초** (기존 설정, 유지 중)

```
1 사이클 = 분석(Flash 1회) + 생성(Imagen3 or Pollinations 1회) + 학습
→ 사이클당 Flash 최소 2회 소모
→ 45초 쿨다운 → 분당 최대 1.3 사이클
→ RPM 사용량: ~2.6/9 → ✅ 안전 범위
```
