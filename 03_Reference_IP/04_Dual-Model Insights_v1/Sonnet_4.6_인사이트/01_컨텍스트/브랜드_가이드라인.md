# 소시안(Socian) 브랜드 가이드라인

> 최초 생성: 2026-04-16 | 출처: 공식 웹사이트 분석

---

## 1. 브랜드 정체성 (Brand Identity)

### 브랜드명
- **공식 명칭**: 소시안 (Socian)
- **영문 표기**: Socian
- **카테고리**: AI 기반 소셜/커뮤니티 분석 솔루션 (B2B SaaS)

### 브랜드 미션
> "데이터로 사람을 이해하고, 커뮤니티를 이해한다"

### 브랜드 포지셔닝
- 타겟 고객: 기업 마케터, 브랜드 매니저, 리서치 담당자
- 포지션: 신뢰할 수 있는 **AI 소셜 인텔리전스 파트너**
- 차별점: 한국어 기반 소셜 데이터에 특화된 분석력

---

## 2. 브랜드 보이스 & 톤 (Brand Voice & Tone)

| 속성 | 설명 |
|------|------|
| **전문적** | 데이터 기반의 명확하고 신뢰 있는 커뮤니케이션 |
| **친근함** | B2B이지만 딱딱하지 않은, 쉽게 설명하는 어조 |
| **혁신적** | AI와 기술을 앞세우되 과하게 기술적이지 않음 |
| **간결함** | 핵심만 전달, 복잡한 설명 지양 |

### 커뮤니케이션 원칙
- ✅ 사용자가 얻을 **가치(Value)** 중심으로 설명
- ✅ 복잡한 기술 개념을 **쉬운 언어**로 번역
- ✅ 수치와 사례 기반의 **사실적 표현** 사용
- ❌ 과도한 기술 용어 남발 지양
- ❌ 과장된 마케팅 수식어 지양

---

## 3. 브랜드 핵심 가치 (Core Values)

1. **데이터 신뢰성** — 정확하고 검증된 분석 데이터 제공
2. **사용자 중심** — 직관적이고 쉬운 UX 설계
3. **한국시장 특화** — 국내 소셜 플랫폼 및 언어 심층 지원
4. **파트너십** — 단순 툴이 아닌 함께 성장하는 비즈니스 파트너

---

## 4. 브랜드 금지 사항

- 브랜드명 **"소시안"과 "Socian"** 혼용 시, 공식 표기(소시안/Socian) 준수
- 로고 색상 임의 변경 금지
- 다른 브랜드와 비교 비방식 표현 금지
- 비공식 약어(SCN 등) 사용 금지
