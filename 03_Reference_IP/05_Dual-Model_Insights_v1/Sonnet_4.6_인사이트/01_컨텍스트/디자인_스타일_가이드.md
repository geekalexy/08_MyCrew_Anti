# 소시안(Socian) 디자인 스타일 가이드

> 최초 생성: 2026-04-16 | 출처: 공식 웹사이트 시각 분석

---

## 1. 컬러 시스템 (Color System)

### 주요 색상 (Primary)
| 역할 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| Primary Blue | 소시안 블루 | `#3B7BF5` | CTA버튼, 강조 텍스트, 핵심 UI |
| Background | 화이트 | `#FFFFFF` | 페이지 배경 |
| Dark Navy | 다크 네이비 | `#0F1F4B` | 푸터, 섹션 배경 |

### 보조 색상 (Secondary)
| 역할 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| Light Gray BG | 연회색 | `#F5F7FA` | 섹션 구분 배경 |
| Text Dark | 다크 그레이 | `#1A1A2E` | 본문 텍스트 |
| Text Medium | 미디엄 그레이 | `#6B7280` | 서브 텍스트, 설명 |
| Border | 라이트 그레이 | `#E5E7EB` | 카드 보더, 구분선 |

### 컬러 사용 원칙
- 배경은 **화이트 또는 연회색** 위주 → 콘텐츠 가독성 최우선
- CTA(행동 유도) 요소에는 **소시안 블루** 단일 사용으로 집중도 유지
- 푸터와 강조 섹션에는 **다크 네이비** 활용으로 신뢰감 형성

---

## 2. 타이포그래피 (Typography)

### 폰트 패밀리
- **한국어**: Noto Sans KR (또는 Pretendard)
- **영문/숫자**: Inter, Noto Sans

### 텍스트 스케일
| 레벨 | 크기 | 굵기 | 용도 |
|------|------|------|------|
| Hero Title | 40–48px | 700 (Bold) | 히어로 섹션 메인 헤드라인 |
| H2 | 28–32px | 700 | 섹션 제목 |
| H3 | 20–24px | 600 | 서브 섹션, 카드 제목 |
| Body | 15–16px | 400 | 본문 텍스트 |
| Caption | 13–14px | 400 | 보조 설명, 레이블 |

### 텍스트 정렬 원칙
- 히어로 섹션: **중앙 정렬**
- 본문 콘텐츠: **좌측 정렬**
- 카드 내부: **좌측 정렬**

---

## 3. 레이아웃 & 스페이싱 (Layout & Spacing)

### 그리드 시스템
- **최대 너비**: 1200px (컨테이너)
- **컬럼**: 12 Column Grid
- **거터**: 24px
- **모바일 분기점**: 768px

### 섹션 스페이싱
- 섹션 간 수직 패딩: `80px ~ 120px`
- 카드 내부 패딩: `24px ~ 32px`
- 요소 간 간격: `16px / 24px / 40px` (8px Base 단위)

---

## 4. 컴포넌트 스타일 (Components)

### 버튼 (Button)
```
Primary CTA:
  - Background: #3B7BF5
  - Text: #FFFFFF
  - Border Radius: 8px
  - Padding: 12px 28px
  - Font: 16px / 600

Secondary:
  - Background: transparent
  - Border: 1px solid #3B7BF5
  - Text: #3B7BF5
```

### 카드 (Card)
```
- Background: #FFFFFF
- Border: 1px solid #E5E7EB
- Border Radius: 12px
- Box Shadow: 0 2px 12px rgba(0,0,0,0.06)
- Padding: 24–32px
```

### 아이콘
- 스타일: **아웃라인(Outline)** 위주
- 크기: 20px / 24px 기준
- 색상: 소시안 블루(강조) 또는 미디엄 그레이(일반)

---

## 5. 이미지 & 일러스트레이션 (Imagery)

### 스크린샷/목업
- **디바이스 목업** 활용 (스마트폰 또는 대시보드 화면)
- 배경은 연회색 또는 화이트
- 실제 서비스 화면 스크린샷 적극 활용 → 신뢰도 강화

### 패턴/배경 요소
- 블루 계열 그라데이션 도형 (하단/우측 배치)
- 기하학적 도트/체인 패턴으로 기술적 느낌 강조
- 과도한 장식 지양 → 심플함 유지

---

## 6. 모션 & 인터랙션 (Motion)

- 인터랙션: **Ease In-Out**, 200–300ms
- 호버 효과: 버튼 살짝 어둡게(Darken 10%), 카드 Shadow 강화
- 페이지 전환: 페이드 인(Fade In), 슬라이드업(Slide Up) 계열
