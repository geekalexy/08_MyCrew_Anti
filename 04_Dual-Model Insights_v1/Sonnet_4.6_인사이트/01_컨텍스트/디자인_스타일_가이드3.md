## 1. 타이포그래피 (Typography)

### 공식 확인 폰트 (CSS 소스 기준)
- **한국어 메인 폰트**: `SUIT` ⭐ *(웹사이트 CSS에서 직접 확인)*
- **대체 폰트**: Noto Sans KR, Pretendard
- **영문/숫자**: Inter, 시스템 폰트

### 텍스트 스케일
| 레벨 | 크기 | 굵기 | 용도 |
|------|------|------|------|
| Hero Title | 40–48px | 800 (ExtraBold) | 히어로 메인 헤드라인 |
| H2 | 28–32px | 700 | 섹션 제목 |
| H3 | 20–24px | 600 | 서브 섹션, 카드 제목 |
| Body | 15–16px | 400 | 본문 텍스트 |
| Caption | 13–14px | 400 | 보조 설명 |

### 텍스트 스타일 원칙
- **line-height**: 170% (CSS 확인값)
- **letter-spacing**: 0.32px (CSS 확인값)
- **기본 텍스트 색상**: `#000000` (CSS 확인값)

## 3. 레이아웃 & 스페이싱

- **최대 너비**: 1200px (컨테이너)
- **컬럼**: 12 Column Grid / 거터 24px
- **모바일 분기점**: 900px (CSS 미디어 쿼리 확인값: `min-width: 900px`)
- **로고 크기**: 모바일 120×27px → 데스크톱 140×32px (CSS 확인)
- **섹션 수직 패딩**: 80–120px
- **카드 내부 패딩**: 24–32px

---

## 4. 컴포넌트 스타일

### 버튼
```
Primary CTA:
  Background: #3B7BF5 | Text: #FFFFFF
  Border Radius: 8px | Padding: 12px 28px | Font: 16px/800

Secondary (카카오 문의):
  카카오 브랜드 컬러 활용 가능
```

### 카드
```
Background: #FFFFFF | Border: 1px solid #E5E7EB
Border Radius: 12px | Shadow: 0 2px 12px rgba(0,0,0,0.06)
Padding: 24–32px
```

### 가격 카드 (Pricing)
- 플랜별 구분 카드 레이아웃
- 프로모션 배지/할인율 표시 (`51% OFF` 등)
- 추천 플랜은 파란색 Border 또는 배경 강조

## 6. SNS/블로그 콘텐츠 스타일

### 네이버 블로그 포스팅
- **제목**: `핵심기능/이슈 + 부연설명 + 이모지` 패턴
  - 예: `클릭율 높이는 신기능 "리타겟팅DM" 출시 🎯`
- **본문 구조**: 공지 → 혜택 → 방법 → 유의사항 → CTA
- **이모지 섹션 구분**: 📅 🎁 ✍️ 📌 활용
- **강조 표현**: 핵심 수치/혜택은 별도 단락 구분

### 인스타그램 콘텐츠
- 블루+화이트 배경의 카드뉴스 포맷 적합
- 핵심 메시지 1개 집중
- 필수 해시태그 포함 (브랜드_가이드라인3.md 참조)
