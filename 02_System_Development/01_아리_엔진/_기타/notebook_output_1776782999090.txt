[Scene 1] (7초)
텍스트: AI 코딩, 혹시 이런 문제 겪고 계신가요? 똑똑한 AI는 비싸고, 무료 AI는 코드가 엉망이고... 뒤치다꺼리 하느라 더 바쁘셨죠?
비주얼 기획: 화면 분할: 한쪽에는 비싼 AI 서비스 아이콘과 '고가' 텍스트, 다른 한쪽에는 엉망으로 꼬인 코드 이미지와 '무료 AI의 한계' 텍스트. 중앙에 물음표 아이콘과 함께 'AI 코딩, 아직도 어렵나요?'라는 자막.

[Scene 2] (5초)
텍스트: 이제 걱정 마세요! Claude Code와 함께라면, 아이디어만으로 실제 작동하는 업무 자동화 서비스를 누구나 만들 수 있습니다.
비주얼 기획: 매끄러운 애니메이션 전환. 'Claude Code' 로고가 중앙에 나타나고, 주변으로 '아이디어' -> '코드' -> '서비스'로 이어지는 화살표가 그려진다. 밝고 긍정적인 분위기.

[Scene 3] (8초)
텍스트: 이번 영상에서는 Claude Code를 활용해 제안서 생성기를 직접 기획하고 개발, 배포하는 전 과정을 보여드립니다. 설치부터 코드 작성, 그리고 실제 서비스 구현까지!
비주얼 기획: 타임랩스처럼 빠르게 진행되는 화면. 개발 환경 설정, 코드 입력, 웹사이트 디자인, 그리고 최종적으로 '제안서 생성기' 서비스가 작동하는 모습이 순차적으로 나타난다.

[Scene 4] (6초)
텍스트: 특히, 기존 방식보다 토큰 사용량을 3분의 2나 절약하면서 더 완벽한 코드를 만드는 비법도 공개합니다. Max 100달러 요금제로 비용 부담 없이 작업하세요!
비주얼 기획: 그래프 애니메이션: '토큰 사용량' 그래프가 3분의 1로 줄어드는 모습. 옆에는 '월 $100' 요금제 플랜이 강조되어 나타난다. '비용 절감'이라는 키워드 강조.

[Scene 5] (7초)
텍스트: 이 영상 끝까지 보시면, 여러분도 AI 개발의 무한한 가능성을 현실로 만들 수 있습니다. 지금 바로 시작합니다!
비주얼 기획: 빠르게 스크롤되는 코드 화면 뒤로 '무한한 가능성', '현실로'라는 텍스트가 오버랩된다. '지금 시작하세요!'라는 CTA 버튼이 등장하며 다음 장면으로 전환.

[Scene 6] (10초)
텍스트: 첫째, Claude Code 설치 및 개발 환경 완벽 세팅! IDE 활용법부터 API 연동까지, 막힘없이 따라오세요.
비주얼 기획: 화면 녹화: Claude Code 설치 과정, VS Code와 같은 IDE 설정, 간단한 API 키 입력 및 테스트 과정을 상세하게 보여준다. '설치 & 세팅'이라는 챕터 타이틀 등장.

[Scene 7] (12초)
텍스트: 둘째, 제안서 생성기 아이디어 구체화 및 기획! 어떤 기능을 넣을지, 사용자 경험은 어떻게 설계할지 함께 고민해봅시다.
비주얼 기획: 화이트보드에 아이디어를 적고 마인드맵을 그리는 모습, 와이어프레임 또는 목업 디자인 툴을 사용하여 서비스 UI/UX를 설계하는 과정을 보여준다. '기획 & 설계' 챕터 타이틀.

[Scene 8] (15초)
텍스트: 셋째, Claude Code로 핵심 기능 코드 작성! 프롬프트 엔지니어링부터 실제 코드 생성까지, AI와 협업하는 모든 과정을 보여드립니다.
비주얼 기획: Claude Code 채팅 인터페이스와 코드 에디터 화면을 번갈아 보여준다. 사용자가 프롬프트를 입력하면 AI가 코드를 생성하고, 이를 복사하여 에디터에 붙여넣는 과정을 시연한다. '코드 작성 & 생성' 챕터 타이틀.

[Scene 9] (10초)
텍스트: 넷째, 디버깅 및 테스트! 예상치 못한 오류는 어떻게 해결하고, 코드가 제대로 작동하는지 검증하는 방법까지 알려드립니다.
비주얼 기획: 오류 메시지가 뜨는 화면과 이를 수정하는 개발자의 모습. 단위 테스트 코드 실행 장면, 또는 간단한 테스트 케이스를 직접 실행해보는 장면을 보여준다. '디버깅 & 테스트' 챕터 타이틀.

[Scene 10] (8초)
텍스트: 다섯째, 클라우드 배포까지 원스톱! 완성된 서비스를 세상에 공개하는 모든 과정을 쉽고 빠르게 안내합니다.
비주얼 기획: 배포 과정(예: Vercel, Netlify 등)을 간략하게 보여준다. 최종적으로 완성된 '제안서 생성기' 웹사이트가 인터넷에서 작동하는 모습을 보여준다. '배포 & 완료' 챕터 타이틀.

[Scene 11] (5초)
텍스트: 이 모든 과정을 통해 여러분은 반복적인 업무에서 벗어나 창의적인 일에 더 집중할 수 있게 됩니다.
비주얼 기획: 한 사람이 여유롭게 커피를 마시며 아이디어를 떠올리는 모습, 또는 동료들과 즐겁게 회의하는 모습. '업무 효율성 증대'라는 텍스트와 함께 긍정적인 분위기 연출.

[Scene 12] (5초)
텍스트: 지금 바로 Claude Code로 여러분의 아이디어를 현실로 만들어보세요! 더 궁금한 점은 댓글로 남겨주세요.
비주얼 기획: 최종 완성된 '제안서 생성기' 서비스 화면 위에 '지금 바로 시작하세요!'라는 CTA와 함께 구독, 좋아요 버튼 애니메이션. 댓글 창 아이콘과 함께 '궁금한 점은 댓글로!'라는 안내 문구.
