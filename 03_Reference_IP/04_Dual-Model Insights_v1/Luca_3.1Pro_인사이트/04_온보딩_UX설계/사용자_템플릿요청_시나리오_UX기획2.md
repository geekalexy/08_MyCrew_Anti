# 📖 사용자 템플릿 요청 시나리오 & UX 기획 2

본 문서는 사용자가 "템플릿 기획을 시작해달라"고 요청할 때, 귀찮음을 극복하고 **핵심 컨텍스트 자료를 적극적으로 제공하게 만드는 넛지(Nudge)형 하이브리드 UX(채팅+카드)** 기획안입니다.

---

## [기획의 주요 대전제]
1. **귀찮음 극복**: 사용자는 자료 입력 작업이 귀찮으며, 템플릿 폴더 구성의 중요성에 대한 인식이 매우 낮다.
2. **동기화(Sync) 주지**: 템플릿과 컨텍스트는 매우 쉽게 접근 가능해야 하며 항상 최신화(Sync)되어야 함을 끊임없이 상기시켜야 한다.
3. **90% 퀄리티 법칙**: 크루(AI 에이전트)가 만들어내는 결과물 만족도의 90%는 '컨텍스트와 템플릿 세팅'에서 나온다는 것을 유저에게 강하게 주지시킨다.

---

## 🌊 [개선된 버전] 가스라이팅(?)과 편의성을 곁들인 템플릿 수집 워크플로우

### 1. 대화 시나리오 (아리의 뼈 때리는 채팅 인터랙션)
* **UX 디자인 포인트**: 시작부터 패널티(90% 퀄리티 타격)를 고지하되, 타자 칠 필요 없이 클릭과 링크 복사만 하라고 안심시킵니다.

> **유저**: "이번 달 쿠폰 이벤트, 릴스랑 블로그용 템플릿 기획 좀 하나 뽑아봐."
> 
> **아리 (Agent)**: "네, 대표님! 기획안을 뽑기 전에 딱 하나만 여쭤볼게요. 
저희 크루가 가져오는 **기획안 퀄리티의 90%는 '컨텍스트 폴더'와 '템플릿 폴더'가 얼마나 잘 설정되어 있느냐**에서 결정된다는 사실 알고 계셨나요? 빈손으로 시키시면 뻔한 AI 글이 나오지만, 재료만 물려주시면 10년 차 마케터가 쓴 글이 나옵니다. 😎
> 
> 바쁘고 귀찮으신 거 너무 잘 압니다! 글씨 입력하실 필요 없이, 아래 카드에 **'참고할 링크'** 딱 하나 퍼다 주시고 **'연동할 폴더'**만 체크해 주세요. 제가 알아서 폴더 최신화(Sync)까지 싹 맞추고 기획 진행하겠습니다!"

---

### 2. 구체적인 UI 구성 (설득형 수집 카드)
* **UX 디자인 포인트**: 계속해서 유저가 폴더의 존재를 인지하게 만들면서, 입력의 귀찮음을 극단적으로 없앱니다.

**[카드 헤더]**
* **Title**: 🔥 기획 퀄리티 90%를 결정하는 [10초 셋업]
* **Subtitle**: 타자는 안 치셔도 됩니다. 던져주신 자료는 자동으로 우리 회사의 영구적인 '컨텍스트/템플릿 폴더'로 동기화(Sync)됩니다.

**[블록 1 : 핵심 기획 무기 (타이핑 최소화, URL 중심)]**
* **라벨**: 📢 크루가 알아서 읽어올 **'핵심 링크나 파일'**만 쓱 던져주세요.
* **캡션(힌트)**: 길게 기획 의도를 쓰실 필요 없습니다! 가격 사이트, 노션 링크 놔주시면 크루가 다 읽습니다.
* **Placeholder**: `URL 붙여넣기 혹은 파일 드래그&드롭`

**[블록 2 : 🧠 강력한 무기(사내 폴더) 1초 동기화]**
* **라벨**: 🗂️ 기획의 베이스가 될 **최신 컨텍스트**를 연결해 주세요.
* **UI 알림 패널**: `💡 Alert! 템플릿/컨텍스트 폴더가 항상 최신(Sync) 상태여야만, 다음번에 똑같은 품질의 아웃풋을 버튼 한 번에 뽑을 수 있습니다.`
* **체크박스 목록**: (기본적으로 모두 체크된 상태로 제공)
  - [✅] `04_마케팅팀/비즈니스_맥락3.md` (사업 최신 정보 연동)

  - [ ] `(비어 있음) 새로운 템플릿 폴더 동기화하기`

### 3. 대화 시나리오 (제출 직후 채팅 복귀 파트)
* **UX 디자인 포인트**: 폴더 시스템의 위력을 한 번 더 각인시켜, 유저 스스로 폴더 관리에 애정을 가지게 만듭니다.

> **유저 (User)**: *(링크를 붙여넣고 카드 완료 클릭)*
> 
> **아리 (Agent)**: "셋업 즉시 완료! 
대표님이 체크해 주신 `사용자_피드백` 폴더의 VOC 문장들과 방금 주신 `링크`가 완벽하게 싱크로율 100%로 결합 중입니다. 
>
> 잠시 후 기획안이 나오면, **이 성공적인 방법론 자체를 '새로운 템플릿'으로 자동 저장해 둘게요.** 그럼 다음번에는 대표님이 버튼 한 번만 누르셔도 이 퀄리티 그대로 복사되니까요! 1분만 기다려주세요! ⏳"
