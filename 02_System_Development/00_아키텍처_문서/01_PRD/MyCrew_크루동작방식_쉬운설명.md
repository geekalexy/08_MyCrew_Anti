## 1. 크루가 일하는 두 가지 통로

MyCrew AI 크루는 일을 할 때 두 가지 경로 중 하나를 사용합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   대표님이 카드에 일을 맡긴다                                        │
│          │                                                      │
│          ▼                                                      │
│   ┌─────────────┐                                               │
│   │   서버 수신  │                                               │
│   └──────┬──────┘                                               │
│          │                                                      │
│    ┌─────┴──────────────────────────┐                           │
│    │ 어떤 크루냐?                     │                           │
│    └─────┬──────────────────────────┘                           │
│          │                                                      │
│    ┌─────┴──────┐       ┌────────────────┐                      │
│    │Anti-Bridge │       │  Gemini 직접   │                      │
```

## 2. 어떤 크루가 어떤 통로를 쓰나

```
                    ┌─────────────────┐
                    │  일을 맡긴다     │
                    └────────┬────────┘
                             │
          ┌──────────────────┼──────────────────┐
          │                  │                  │
          ▼                  ▼                  ▼
  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐
  │  Anti-Bridge  │  │  Anti-Bridge  │  │    Gemini     │
  │    Sonnet     │  │  Prime/Nexus  │  │  Flash / Pro  │
  └───────┬───────┘  └───────┬───────┘  └───────┬───────┘
          │                  │                  │
          ▼                  ▼                  ▼
   ┌─────────────┐   ┌─────────────┐   ┌─────────────┐
   │  lily (릴리) │   │  luna (루나) │   │  nova (노바) │
   │  pico (피코) │   │  ollie(올리) │   │  lumi (루미) │
   │             │   │             │   │  ari  (아리) │
   └─────────────┘   └─────────────┘   └─────────────┘
```

**통로 설명:**
- **Anti-Bridge Sonnet** → Antigravity 앱(소넷)이 처리 → Claude Sonnet급 품질
- **Anti-Bridge Prime/Nexus** → Antigravity 앱이 처리 → 동일하게 구독 세션 활용
- **Gemini** → Google API 직접 처리 → 빠른 응답

## 4. 구글 구독 상태에 따른 모델 자동 전환 (nova, lumi)

```
사용자 설정: USER_MODEL_TIER
          │
          ▼
┌─────────────────┐     ┌──────────────────────────────────┐
│  USER_MODEL_    │     │ .env 파일에 한 줄 추가             │
│  TIER = pro     │◀────│ USER_MODEL_TIER=pro               │
└────────┬────────┘     └──────────────────────────────────┘
         │
    ┌────┴────┐
    │         │
    ▼         ▼
┌───────┐  ┌─────────────────────┐
│  pro  │  │  (미설정 / 기본값)   │
└───┬───┘  └──────────┬──────────┘
    │                 │
    ▼                 ▼
┌──────────┐      ┌──────────────┐
│ Gemini   │      │ Gemini Flash │
│ Pro      │      │ (빠르고 무난)  │
│(높은 품질) │      │              │
└──────────┘      └──────────────┘
```

**언제 Pro를 쓰나?**
- Google One AI Premium (월 2만원대) 구독 중
- Gemini Ultra 구독 중
- → `.env` 파일에 `USER_MODEL_TIER=pro` 한 줄 추가하면 자동 전환

## 5. 현재 상태 (솔직한 현황)

```
크루      │ 담당 통로          │ 지금 실제 작동 여부
──────────┼────────────────────┼──────────────────────────────
lily      │ Anti-Bridge Sonnet │ ⚠️  브릿지 감시자 미완성
          │                    │    → 5분 대기 후 Gemini로 대체
pico      │ Anti-Bridge Sonnet │ ⚠️  동일
luna      │ Anti-Bridge Nexus  │ ⚠️  동일
ollie     │ Anti-Bridge Prime  │ ⚠️  동일
nova      │ Gemini Flash/Pro   │ ✅  정상 작동
lumi      │ Gemini Flash/Pro   │ ✅  정상 작동
ari       │ Gemini Flash       │ ✅  정상 작동
```

**남은 작업**: `ariDaemon.js`에 브릿지 파일 감시 기능 구현  
→ 완료되면 lily/pico/luna/ollie 모두 Antigravity(소넷) 구독으로 직접 처리됨
