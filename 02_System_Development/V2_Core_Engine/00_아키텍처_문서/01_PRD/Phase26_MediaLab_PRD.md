## 1. 경쟁 서비스 리서치 요약

### 레퍼런스 서비스 분석

| 서비스 | 핵심 UX 패턴 | 우리가 배울 것 |
|--------|------------|--------------|
| **Runway** | Pro-grade 스튜디오 레이아웃, 정밀 컨트롤 | 전문가용 느낌의 패널 배치 |
| **HeyGen** | 템플릿 → 스크립트 → 아바타 선택 3단계 | 채널 선택 → 자동화 → 리뷰 흐름 |
| **OpusClip** | Virality Score UI, 클립 한 줄 리뷰 | AI 큐레이션 스코어 시각화 |
| **InVideo AI** | 프롬프트 퍼스트, 네러티브 드리븐 | 채널 목적 중심 시작 |
| **Submagic** | 캡션 스타일 프리셋 갤러리 | TTS 버전 선택 UI |
| **Pictory** | 워터마크 + Usage Limit 프리미엄 전환 | Paywall 전략 |

### 공통 디자인 패턴 (상용 서비스 표준)

1. **채도 낮은 다크 배경** — 브랜드 1색만 강조
2. **진행 상태 투명성** — 각 에이전트 단계를 실시간으로 보여줌
3. **고정 좌측 패널 + 중앙 캔버스 + 우측 컨트롤** 3-column 레이아웃
4. **컨텍스추얼 페이월** — 기능을 시도할 때 잠금 UI 표시
5. **소셜 프루프 배지** — "이번 달 xxx건 생성됨" 등 숫자 노출

## 2. MyCrew MediaLab 브랜드 컬러 시스템 (통일)

### 현재 문제
- 씬 배지 5색(빨강/주황/파랑/보라/초록) 동시 표시 → 난잡함
- TTS 버전 색 3종 별도 운영
- 채널별 테마 컬러가 UI 전체를 침범

### 해결: 2+1 컬러 시스템

```
Primary:   #4ade80  (MyCrew Green — 완료, 성공, 메인 CTA)
Danger:    #ef4444  (FAIL 판정, 포커스 타겟팅에만 사용)
Neutral:   #94a3b8  (보조 텍스트, 배지 라벨)

Background:
  Base:     #09090b  (최하층)
  Surface:  #111118  (카드/패널)
  Elevated: #18181f  (호버, 선택 상태)

Text:
  Primary:  #f1f5f9
  Secondary: #94a3b8
  Muted:    #475569

Border:    #1e1e2e (기본)
```

### 씬 배지 단순화
```
변경 전: 🔥 HOOK (빨강), ⚠️ PROBLEM (주황), 📊 PROOF (파랑), ⚡ CLIMAX (보라), 🔔 CTA (초록)
변경 후: 모두 배경 #1e1e2e + 텍스트 #94a3b8 — 이모지만 색 역할
         포커스 선택 시만 #ef4444 테두리 사용
```

## 4. YouTube Autopilot 5단계 위저드 플로우 (VideoLab 개편)

### 플로우 지도

```
[1.채널선택] → [2.파이프라인] → [3.큐레이션검수★] → [4.리뷰스튜디오] → [5.발행]
```

★ **3단계 "큐레이션 검수"가 핵심 신규 화면** — 현재 완전히 빠진 검증 루프 해결

### 각 단계 Back 버튼 동작 (버그 수정)

```js
const VIEW_FLOW = ['CHANNELS', 'AUTOMATING', 'CURATION', 'REVIEW', 'RESULT'];

const handleBack = () => {
  const curr = VIEW_FLOW.indexOf(viewState);
  if (curr <= 0) setCurrentView('projects'); // 대시보드 복귀
  else setViewState(VIEW_FLOW[curr - 1]);    // 이전 단계
};
```
