# Agent Control End-to-End Testing

사용자가 텔레그램 미니앱에서 특정 에이전트 제어 명령을 내렸을 때, 해당 명령이 백엔드를 통해 목표 에이전트에게 정확히 전달되고 예상된 결과를 반환하는지 검증하는 종단간(End-to-End) 테스트 시나리오를 설계하고 자동화한다. Mock Agent를 활용하여 테스트 환경을 구축한다.
