# 🎨 디자인 스타일 가이드 (Design Style Guide)

## 1. 컬러 팔레트 (Color Palette)
- **Primary Color (핵심 브랜드 색상)**: `Vibrant Blue` (활기차고 신뢰감을 주는 선명한 블루 계열)
  - 적용처: CTA (Call-to-Action) 버튼, 핵심 강조 텍스트, 메인 배너 및 일러스트 배경.
- **Background Color**: `Pure White` & `Light Grayish Blue` (아주 연한 푸른빛의 쿨 그레이)
  - 적용처: 콘텐츠 블록을 깔끔하게 분리하고 시각적 안정감을 주기 위한 바탕색.
- **Accent Color**: `Yellow / Orange` (포인트 컬러)
  - 적용처: 배지(Badge), 알림 스파크, 핵심 기능을 돋보이게 하는 리듬감 있는 포인트 요소.

## 2. 타이포그래피 (Typography)
- **서체 무드**: 모던하고 가독성이 뛰어난 네오그로테스크 / 고딕 계열 (예: Pretendard 등).
- **계층 구조 (Hierarchy)**:
  - **헤드라인 (Title)**: 굵은(Bold) 웨이트로 크고 명확하게 타이틀을 배치하여 시선을 사로잡음. 핵심 단어는 Primary Blue로 하이라이트.
  - **본문 (Body)**: 여유 있는 자간과 행간으로 텍스트 정보의 가독성 극대화.

## 3. UI 컴포넌트 & 도형 (Shapes)
- **둥근 모서리 (Rounded Corners)**:
  - 프레임, 카드 UI, 버튼 등에 곡선을 적용하여 친근함과 부드러운 사용자 경험 제공.
- **카드형 레이아웃 (Card UI)**:
  - 인플루언서 프로필, 기능 설명 등에 미세한 그림자(Drop Shadow)가 들어간 카드 UI를 사용하여 배경과 분리 및 입체감 확보.
- **다이내믹 셰이프 (Dynamic Geometric Patterns)**:
  - 하단 푸터 및 주요 배너의 배경에서 기하학적 곡선, 겹치는 반투명 원형 패턴 등을 활용하여 '연결', '유연성', '확장'의 브랜드 아이덴티티를 그래픽으로 표현.

## 4. 이미지 및 에셋 (Imagery)
- **사용자(인물) 실사 이미지**: 자연스러운 라이프스타일이나 작업 환경을 보여주는 고품질 인플루언서 화보 활용.
- **목업 (Mock-up) 기반 UI 요소**: 단순 텍스트 설명이 아닌, 실제 서비스(앱) 화면이나 채팅UI가 들어간 목업 이미지를 띄워 서비스의 실재감과 시스템의 편리함을 강조.
