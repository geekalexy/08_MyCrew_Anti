# 🏆 [최종 보고서] 컨텍스트 및 템플릿 수집 워크플로우 종합 결과

본 문서는 오늘 진행된 총 5단계의 '비즈니스 컨텍스트 및 템플릿 수집 실증 실험'의 최종 결과물이자, MyCrew 에이전트 시스템을 구동하는 대원칙입니다.

---

## 1. 🌊 전 과정 워크플로우 정의 (7-Day 마스터플랜)
팀리더 에이전트(아리 등)가 유저를 온보딩시키고 실무 템플릿 공간을 채우는 전 과정은 **'7일 타이밍 집중화 전략'**으로 작동합니다.

*   **[섹션 1. 컨텍스트 수집 (D-Day ~ D+2)]**
    *   **D-Day**: 대시보드 첫 진입 시 UI 팝업. "딱 한 개의 홈페이지 링크만 던져달라"고 유도. 에이전트가 야간 백그라운드 파싱.
    *   **D+1**: 에이전트가 "핵심 고객은 OO이고, 문제는 XX죠?"라고 단답형 컨펌 요구 ➡ **컨텍스트 100% 확보.**
*   **[섹션 2. 템플릿 공유 및 대응 (D+3 ~ D+7)]**
    *   **D+3/D+4**: 첫 마케팅 쿠폰 이벤트 기획 넛지. "타자 치지 말고 타사 링크 하나와 체크박스(기존 컨텍스트 연동)만 켜라"며 UI 카드 입력 유도.
    *   **D+5**: 에이전트 크루가 파싱하여 완벽한 초안 도출. 유저가 만족 시 사이드바에 '영구 템플릿'으로 핀(Pin) 고정. ➡ **템플릿 확보.**
    *   **D+7**: 고정된 템플릿을 바탕으로 주간 자동 보고서 발행 (자율 주행 시작).

---

## 2. 📁 폴더 생성 타이밍 및 하위 폴더 매핑
*   **생성 타이밍 1 (D-Day)**: 사용자가 첫 홈페이지 링크를 제출한 직후, `04_마케팅팀/컨텍스트/` 폴더가 자동 생성되며 핵심 3대 파일(`비즈니스_맥락.md`, `브랜드_가이드라인.md`, `디자인_스타일_가이드.md`) 매핑.
*   **생성 타이밍 2 (D+5)**: 에이전트가 도출한 첫 캠페인 초안을 사용자가 승인(Pin)하는 순간, `04_마케팅팀/템플릿/` 폴더 산하에 `주간_보고서_템플릿.md`, `쿠폰이벤트_종합기획안.md` 등으로 자동 아카이빙.

---

## 3. 🎯 에이전트의 구체적 정보 수집 방법 (UX/UI 전략)
사용자의 피로도(귀찮음)를 극복하기 위한 **'하이브리드 넛지(Nudge) 수집법'** 채택.
*   **수집 철학**: "기획안 퀄리티의 90%는 이 셋업에서 나온다"는 패널티(가스라이팅) 요소를 강조하여 제출 명분 확보.
*   **대화방식 (Chat)**: 스무고개가 아닌, 직관적인 예시를 포함한 단답형 질문. (예: "지갑 열 한 사람 누군가요? 어떤 짜증나는 일을 해결해주시나요?")
*   **카드방식 (Card UI)**: 타이핑(Textarea)을 최소화. `URL 링크 폼 2매` + `사내 폴더 연결 체크박스` + `파일 드래그 앤 드롭`으로만 구성된 초간편 **'10초 셋업 카드'** 활용.

---

## 4. 🤖 팀 전원 학습 시스템 및 워크플로우 (Collaboration)
에이전트 각자가 헤매지 않도록, `04_마케팅팀/컨텍스트/` 폴더가 모든 실무의 **단일 진실 공급원(SSOT)**으로 작용합니다.

1.  **전략 크루**: 사용자가 입력한 링크 + 기존 컨텍스트 폴더를 불러와 목표(이벤트 할인율 등) 수립.
2.  **카피라이팅 크루**: 컨텍스트 내의 '사용자 피드백(VOC)' 폴더를 RAG로 직접 읽고, 릴스 대본 및 블로그 본문 타겟팅 후킹 문구 생성.
3.  **디자인/영상 크루**: '디자인_스타일_가이드.md'의 컬러 팔레트(Socian Blue) 조건값만 읽어와 캐러셀/썸네일 즉시 렌더링.
4.  **퍼포먼스 크루 (★아리)**: 인스타 자동화 DM 키워드 세팅 및 시스템 배포. (가장 중요)

    > 타자 치실 필요 없습니다! 아래 카드에 **대표님 비즈니스의 '가장 중요한 링크(기능/가격)'** 하나만 툭 붙여넣어 주세요. 나머지는 제가 싹 다 파싱해서 각 크루들의 뇌(폴더)에 동기화시키겠습니다!"*
