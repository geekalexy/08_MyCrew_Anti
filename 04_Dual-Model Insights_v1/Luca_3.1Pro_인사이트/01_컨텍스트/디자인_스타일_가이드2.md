# 🎨 디자인 스타일 가이드 (Design Style Guide)

## 1. 컬러 팔레트 (Color Palette)
- **Primary Color (핵심 색상)**: `Vibrant Blue` (활기차고 신뢰감을 주는 블루)
  - 웹사이트의 배너, 버튼 및 네이버 블로그 스킨, 썸네일 테마 컬러로 일관성 있게 사용.
- **Background Color**: `Pure White` & `Light Grayish Blue`
  - 복잡한 정보를 쾌적하게 보여주기 위한 여백용 컬러.
- **Accent Color**: `Yellow / Orange` (포인트 컬러)
  - 배지, 강조 텍스트, UI 스파크 포인트 요소.

## 2. 타이포그래피 (Typography)
- **서체 무드**: 깔끔한 고딕 계열 (Pretendard 등). 
- **블로그 포스팅 규칙**: 가독성을 높이기 위해 모바일과 데스크톱 모두 쾌적한 폰트 크기 및 행간 유지 필수.

## 3. UI 컴포넌트 및 그래픽 (Graphic Elements)
- **둥근 모서리 (Rounded Corners)**
- **카드형 레이아웃 (Card UI)**: 인플루언서 프로필 리스트, 결과 데이터 등은 살짝 그림자가 들어간 얕은 심도의 카드형으로 표현.
- **다이내믹 셰이프**: 반투명한 흰색 원형 및 곡선 등이 파란색 메인 배경 위에서 부드러운 연결(네트워크)감을 줌.

## 4. 이미지 무드 (Imagery)
- 실사 바탕의 깔끔한 라이프스타일 화보(앱/플랫폼을 사용하는 모습).
- 기능 설명 시에는 단순히 글로 나열하지 않고 플랫폼/대시보드의 Mock-up 이미지를 함께 배치하여 신뢰감 고취.
