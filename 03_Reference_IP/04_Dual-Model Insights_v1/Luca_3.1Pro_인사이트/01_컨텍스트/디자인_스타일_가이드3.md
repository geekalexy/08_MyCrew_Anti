# 🎨 디자인 스타일 가이드 (Design Style Guide)

## 1. 컬러 팔레트 (Color Palette)
- **Primary Color (핵심 브랜드 컬러)**: `Socian Blue` (소시안 특유의 신뢰감과 기술력을 보여주는 선명한 블루 색상). 
  - 웹사이트의 구매/무료시작 CTA 버튼 및 블로그, 가이드북의 링크 텍스트에 공통 적용.
- **Background Color**: `Clean White` & `Light Gray` 
  - 깃북 문서와 대시보드의 가독성을 위한 깔끔하고 하얀 여백.

## 2. 타이포그래피 (Typography)
- **서체**: `SUIT` 또는 본고딕(`Noto Sans KR`). 
- **위계(Hierarchy)**: 가격 페이지 플랜명(Free, Pro 등) 및 핵심 자동화 기능 타이틀은 볼드하게 처리하여 첫눈에 구조가 읽히도록 세팅.

## 3. UI 컴포넌트 및 아이콘 시스템 (Iconography)
*(GitBook 가이드 구조에 기반한 아이콘 활용 원칙)*
- **기능별 픽토그램 매핑**: 딱딱한 기능 설명 대신 아이콘을 활용하여 사용자 친화성 극대화.
  - DM/대댓글: ✉️ Envelope 아이콘
  - 통계 대시보드: 📊 Chart Column 아이콘
  - 릴스 업로드: 📤 Up-from-bracket 아이콘
  - 이벤트 추첨: 🎉 Party-horn 아이콘
- **친화적 UI 바운더리**: 라운드 테두리(Rounded border) 버튼 모듈을 통일되게 사용하여 전방위적 친근함 조성.

## 4. 커스토머 저니(Customer Journey) 시각화
- 텍스트 가이드 뿐만 아니라, 인스타그램 연동 시 나타나는 '실제 화면 캡처(권한 팝업 등)'와 '시크릿 모드 단축키(Ctrl+Shift+N)' 등을 시각적인 Block-quote나 박스 형태로 눈에 띄게 디자인하여 사용자 이탈 최소화.
