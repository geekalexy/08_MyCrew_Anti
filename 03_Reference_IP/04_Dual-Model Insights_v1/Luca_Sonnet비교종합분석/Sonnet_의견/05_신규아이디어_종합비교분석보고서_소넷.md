### 아이디어 1: VOC 카피 추출 스킬 (`/VOC_카피추출`)

**내용**: `사용자피드백` 폴더의 실제 고객 불만·칭찬 텍스트를 RAG로 읽어
10개를 즉시 추출해 릴스 후킹 카피 재료로 제공

**왜 중요한가**:

- 소시안 `06_소시안자료/사용자피드백` 폴더와 직접 연동
- 이 스킬 하나가 콘텐츠 품질의 차원을 바꿈

**소넷 평가**: ⭐⭐⭐⭐⭐ — **즉시 채택 필요**

### 아이디어 2: 사이드바 핀(Pin) 유도 CTA

**내용**: 카드 완료 후 "이 컨텍스트/템플릿 폴더를 사이드바 최상단에 고정할까요?" CTA

**왜 중요한가**:
- 폴더가 보여야 업데이트한다 — 눈에 안 보이는 폴더는 존재하지 않는 것
- 습관 형성의 핵심은 접근성
- 대시보드 UI 기능으로 구현 가능

**소넷 평가**: ⭐⭐⭐⭐⭐ — **온보딩 UX에 필수 추가**

### 아이디어 3: 폴더 자동 생성 2-타이밍 설계

**내용**:
- D-Day: 첫 URL 제출 → `컨텍스트 폴더` 자동 생성
- D+5: 사용자 초안 승인(Pin) → `템플릿 폴더` 자동 아카이빙

**왜 중요한가**:
- 사용자가 "저장" 버튼 누르지 않아도 폴더 구조가 완성됨
- SSOT 구축이 자동화됨 — 사용자 행동 없이 시스템이 알아서 생성

**소넷 평가**: ⭐⭐⭐⭐⭐ — **백엔드 핵심 로직으로 채택**

### 아이디어 4: 스노우볼 현황 보고 (`/스노우볼_현황보고`)

**내용**: 캠페인 론칭 D+3, D+7에 댓글 전환율·블로그 리뷰 수 자동 크롤링
→ `주간_보고서_템플릿.md` 양식으로 대표에게 자동 브리핑

**왜 중요한가**:
- 마케팅의 결과를 자동으로 측정·보고
- 대표가 요청 안 해도 자율적으로 KPI 추적
- "자율주행" 상태의 핵심 구성 요소

**소넷 평가**: ⭐⭐⭐⭐ — **L3 Performance 필수 스킬**

### 아이디어 5: /트리거_자동화세팅 (소시안 백엔드 연동)

**내용**: 기획된 이벤트 로직(댓글 키워드 → DM 자동 발송)을
소시안 실제 API에 꽂아 넣어 자동화 즉시 배포

**왜 중요한가**:
- MyCrew가 기획뿐 아니라 **실제 실행**까지 담당
- 소시안 서비스 자체와 MyCrew가 연결되는 핵심 스킬

**소넷 평가**: ⭐⭐⭐ — **장기 로드맵 우선순위 상위**

### 아이디어 6: CPO·CQA·CHRO 3직책 분리

**내용**: C레벨을 하나의 CMO로 뭉치지 않고
기능별로 3개 직책 에이전트로 분리 운영

**왜 중요한가**:
- 각 직책이 전문화 → 에이전트 시스템 프롬프트가 간결해짐
- 하나의 에이전트에 역할이 섞이면 판단 혼란 발생 (역할 혼란 문제 해결)
- CPO = 분배, CQA = 품질, CHRO = 인사 → 각자 명확한 KPI

**소넷 평가**: ⭐⭐⭐⭐ — **Phase 2 PM 분리 시 채택**
