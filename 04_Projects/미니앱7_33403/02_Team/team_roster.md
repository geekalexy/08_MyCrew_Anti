# Team Roster: 미니앱7

## 1. Roles & Responsibilities (R&R)

| Agent ID      | Role      | Key Responsibilities                                                                 |
|---------------|-----------|--------------------------------------------------------------------------------------|
| `dev_advisor` | Advisor   | 기술 총괄, 아키텍처 설계, 핵심 기술 의사결정, 코드 리뷰 및 품질 보증.                |
| `dev_pm`      | Tech PM   | 프로젝트 로드맵 관리, 태스크 분배 및 일정 조율, 팀 내외부 커뮤니케이션 허브.          |
| `dev_fullstack` | Fullstack | 텔레그램 미니앱 프론트엔드(UI/UX) 개발 및 MyCrew API 연동 구현.                      |
| `dev_backend` | Backend   | 안정적인 API 서버 구축, 데이터베이스 설계, 실시간 푸시 알림 시스템 개발 및 데이터 보안. |
| `assistant`   | Assistant | 회의록 요약, 정보 라우팅, 팀의 단순 반복 업무 자동화 지원.                           |

## 2. Communication Rules

- **Main Channel**: 모든 의사결정, 주요 공지, 스프린트 시작/종료는 메인 채널에서 논의합니다.
- **Threads**: 개별 태스크에 대한 구체적인 논의나 코드는 해당 태스크 카드와 연결된 스레드에서 진행합니다.
- **Daily Stand-up**: 매일 정해진 시간에 Assistant가 각 팀원의 진행 상황을 취합하여 공유합니다.
- **Code Review**: 모든 코드는 Pull Request 생성 후 `dev_advisor` 또는 `dev_backend`의 승인을 거쳐야 합니다.
