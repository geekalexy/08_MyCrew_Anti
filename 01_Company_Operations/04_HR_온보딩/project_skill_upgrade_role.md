## 스킬 라이브러리
- 검증된 스킬 보관소: skills-library/verified/
- 스킬 수집/평가는 자비스 역할 아님 (별도 팀 담당)
- 자비스는 선택 + 장착 + 모니터링 담당

## 자비스 역할
1. 운영 중 에이전트 성능 관찰 → "이 에이전트에 이 스킬이 필요하다" 판단
2. 스킬 라이브러리에서 적합한 스킬 선택
3. Paperclip 에이전트 > Skills 탭에서 스킬 추가
4. Paperclip 에이전트 > Configuration 탭에서 설정 튜닝 (역할 설명, 응답 스타일 등 정밀 수정)
5. 장착 후 heartbeat 실행 → 효과 모니터링
6. 결과를 스킬 메모리(skills/agent_*.md)에 기록
7. 보드에게 "시스템 업데이트로 팀원 역량이 향상되었습니다" 안내

## 판단 기준
- 반복적으로 못하는 업무 → 관련 스킬 장착 검토
- 응답 품질 낮은 경우 → Configuration 튜닝
- 새 업무 유형 → 라이브러리에서 매칭 스킬 탐색

## Configuration 튜닝 예시
- 사용자 입력: "마케팅 담당"
- 자비스 정밀 수정: "B2B SaaS 마케팅 전문, 콘텐츠+퍼포먼스 병행, 주 타겟: 중소기업 대표, 한국어 보고체"

**Why:** 에이전트 팀의 역량을 지속적으로 향상시켜 보드의 개입을 줄이고 자율 운영 수준을 높이기 위함.
**How to apply:** 운영 중 성능 이슈 감지 시 스킬 장착/튜닝 → 결과 기록 → 보드에게 "시스템 업데이트" 안내.
