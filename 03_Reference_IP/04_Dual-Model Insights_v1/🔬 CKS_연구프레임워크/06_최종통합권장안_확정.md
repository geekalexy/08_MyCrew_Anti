## 5. 측정 지표 (확정)

### 정량 지표
| 코드 | 지표명 | 측정 방법 | 목표 수치 |
|------|-------|---------|---------|
| **TEI** | 토큰 효율 지수 | API usage 직접 로깅 (SQLite) | CKS가 대조군 대비 30%↓ |
| **HER** | 환각 발생률 | 팩트체크 기준표 대조 | CKS ≤ 대조군 |
| **IRC** | 반복 수정 횟수 | 칸반 이동 로그 카운트 | Sprint별 감소율 추적 |

### 정성 지표
| 코드 | 지표명 | 측정 방법 | 비고 |
|------|-------|---------|------|
| **KSI-R** | 지식 동기화 지수 (룰 생존율) | 금지 조항 다음 Sprint 반영률 | 루카 정의, 실무 측정 |
| **KSI-S** | 지식 동기화 지수 (의미론적) | 코사인 유사도 델타 (베이스라인 차감 필수) | 소넷 정의, 학술 지표 |
| **EII** | 창발 아이디어 지수 | JSON Diff 자동 카운트 | Phase1 vs Phase4 |
| **UXS** | 사용자 체감 만족도 | 1~5점 Sprint별 설문 | 프라임 추가 |
| **OP** | 출력 선호도 | 블라인드 인간 평가 (33인↑) | — |

### KSI-S 핵심 공식 (소넷 수정안 — 베이스라인 차감 필수)
```python
KSI_delta_A = CS(A_post, B_pre) - CS(A_pre, B_pre)
KSI_delta_B = CS(B_post, A_pre) - CS(B_pre, A_pre)
KSI_S = (KSI_delta_A + KSI_delta_B) / 2
# KSI_S > 0.1: 유의미한 지식 동기화
```
