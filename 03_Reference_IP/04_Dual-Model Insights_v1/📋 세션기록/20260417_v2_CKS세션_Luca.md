## 2. 세션 하이라이트 (CEO Insight)
1. **에러를 불만으로 보지 말 것**: 에이전트가 룰을 어기거나 오류를 낼 때마다 유저가 짜증을 내는 기존 UX를 탈피.
2. **Team Building UX 도입**: AI와의 작업 과정 자체를 "신규 팀원과 칸반 보드를 활용해 사내 규정(Ground Rule)을 잡아가는 건강한 조직 문화" 로 격상.
3. **토큰 낭비 방지**: 무한 핑퐁 토론을 지양하고, 작업 종료 후 실행되는 "스프린트 회고" 기능을 통해 다음 작업 시 에이전트의 지능을 토큰 낭비 없이 최고 가성비로 이끄는 프레임워크 고안.
