## 👑 CATEGORY 0: C레벨 (CMO) 전용 스킬

> **역할 정의**: 아리가 접수한 요청을 전문 프롬프트로 정제 후 태스크를 병렬 할당하고,
> 결과물을 비평적 시각으로 1차 리뷰한 다음 통과 시 CEO에게 보고 이관하는 총괄 임원.

### C0-E: 품질 준수 감독 스킬

> **이 섹션의 목적**: 크루가 태스크를 올바른 방식으로 수행하는지
> 즉, "올바른 스킬을 쓰는가" + "필수 파일을 먼저 읽는가" 두 축을 감독한다.

| # | 스킬명 | 설명 | 입력 | 출력 | 상세 |
|---|--------|------|------|------|------|
| **0-12** | **스킬 활용 자율성 감독** | 크루가 명시적 지시 없이도 관련 스킬을 적절히 자율 적용하는지 감독 | 태스크 실행 로그 | 스킬 자율성 평가 리포트 | 예: 지시 없이 해시태그 자동 추가, 컨텍스트 파일 자동 참조 등 |
| **0-13** | **컨텍스트/템플릿 이행 감독** | 크루가 태스크 시작 전 컨텍스트 파일과 템플릿 파일을 반드시 읽었는지 이행 여부 감독 + 일간/주간 평가 후 개선 조치 | 태스크 실행 로그 + 파일 접근 기록 | 이행률 리포트 + 개선 지시 | 미이행 시: 경고 → 재작업 지시 → 이행 규칙 강화 |

**0-12 vs 0-13 차이점:**

| 구분 | 0-12 스킬 활용 자율성 감독 | 0-13 컨텍스트/템플릿 이행 감독 |
|------|--------------------------|--------------------------------|
| **감독 대상** | 스킬을 *자율적으로 선택*하는가 | 필수 파일을 *먼저 읽는가* |
| **기준** | 스킬 활용의 적절성·창의성 | 컨텍스트/템플릿 파일 접근 이행 여부 |
| **주기** | 태스크 완료 시 수시 | 일간 + 주간 정기 평가 |
| **미준수 시** | 스킬 보강 피드백 | 재작업 지시 + 규칙 강화 |
| **목표** | 크루 자율성 향상 | 산출물 품질 기반 일관성 확보 |

**0-13 이행 감독 사이클:**
```
[일간 체크] 오늘 완료된 태스크 중 컨텍스트/템플릿 미참조 건 확인
    ↓
    이행 ✅ → 정상 진행
    미이행 ❌ → 해당 크루에 경고 + 재작업 지시

[주간 평가] 이행률 집계 → 이행률 < 80% 시
    ↓
    원인 분석 (파일 접근 어려움? / 프롬프트 누락? / 크루 습관?)
    ↓
    개선 조치: 프롬프트 강화 / SOP 업데이트 / 크루 재교육
```
