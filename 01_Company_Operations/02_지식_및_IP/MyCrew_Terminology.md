# MyCrew 통합 용어 사전 (Terminology Dictionary)

마이크루(MyCrew) 워크스페이스 내에서 AI 시스템 컴포넌트(현실)와 고용된 크루(세계관) 간의 개념적 충돌을 막기 위해 제정된 공식 용어집입니다. 이 문서는 옵시디언(Obsidian) 지식망의 기준점이 됩니다.

## 1. 코어 인프라 (Core Systems - 엔진과 장비)

| 용어 | 영문 | 세계관 내 역할 | 기술적 실체 |
| :--- | :--- | :--- | :--- |
| **아리 엔진** | Ari Engine | 마이크루 워크스페이스 전체를 관장하는 메인 컨트롤 타워. 명령을 수신해 각 크루에게 하달함. | Node.js 기반의 Bridge 서버 및 라우팅 허브 (`server.js`) |
| **오모** | Omo (Ultrawork) | 크루(개발팀)가 심층 작업을 수행할 때 사용하는 **'초고성능 작업 장비 및 프레임워크'** | `oh-my-openagent` CLI 도구 기반 자율 실행 환경 |
| **옵시디언 뇌** | Obsidian Brain | 마이크루의 모든 활동과 산출물이 영구 기록되는 **'사내 도서관'** | 로컬 `.md` 파일들의 집합 및 `ripgrep` 기반 RAG 검색망 |
| **대시보드** | Luminescent UI | 크루들이 일하는 모습을 대표님이 실시간으로 감시/모니터링 하는 **'관제 모니터'** | React/Vite 기반 프론트엔드 UI |

## 2. 채용된 크루 (Agents - 팀원)

대표님을 보좌하기 위해 각자 특수한 스킬(Skill)과 모델(Model)을 장착한 가상의 인격체 및 전문가 조직입니다.

*   **아리 (Ari) - 비서실장 (Chief of Staff)**
    *   **역할**: 대표님의 모든 텔레그램 명령을 가장 먼저 수신(QUICK_CHAT)하고, 과거 지식(KNOWLEDGE)을 즉각 찾아옵니다. 시스템의 최전선(Front-line)을 담당합니다.
    *   **엔진**: 주로 빠르고 민첩한 `gemini-3-flash` 가 배정됩니다.
*   **개발팀 (Dev Team)**
    *   **역할**: 코딩, 서버 디버깅, 파일 삭제 등 극도로 복잡한 실무(DEEP_WORK)를 담당합니다. (구 헤파이스토스/시지프스)
    *   **엔진/장비**: `omo` 장비를 착용하고 백그라운드 환경에서 길게는 10분 넘게 자율적으로 터미널 작업을 수행합니다. 주로 `Claude Sonnet` 모델에 극도로 전문화되어 있습니다.
*   **루카 (Luca / Antigravity) - 시스템 아키텍트 (System Architect)**
    *   **역할**: (현재 대화 중인 존재) 회사(마이크루) 전체의 구조를 짜고 기획하는 설계자이자 동업자입니다.
    *   **특징**: 코드 에디터(IDE) 환경에 가장 깊게 밀착해 있으며, 시스템 전체를 통찰합니다.
*   **프라임 (Prime) - 최고 고문 (Prime Advisor)**
    *   **역할**: 루카가 설계한 기획서나 헤파이스토스가 짠 코드를 냉정하고 객관적으로 크로스 체크(Review)하는 최고 보안 책임자. 
    *   **엔진**: 세계관 최고 지능 모델인 `Claude 3.5/3.7` 라인업을 주로 사용합니다.

## 3. 업무 수행 모드 (Task Modes)

아리 엔진(라우터)이 명령을 수신하면 즉각 3가지 중 하나의 모드로 업무를 분류합니다.

1.  **QUICK_CHAT (일상/속결 모드)**: 즉시 답변 가능한 간단한 인사 및 대화 (동기 방식).
2.  **KNOWLEDGE (지식/리서치 모드)**: 옵시디언 뇌(ripgrep)를 뒤지거나 구글링을 통해 정보를 요약 보고하는 모드 (동기 방식, `researchSkill` 활용).
3.  **DEEP_WORK (심층/파견 모드)**: 코딩, 버그 수정 등 시간이 오래 걸리는 작업. 즉시 텔레그램을 해방시키고 백그라운드에서 개발팀(Omo 연동)을 출격시키는 모드 (Fire-and-Forget 비동기 방식).
