# AI 에이전트 회사 사용 가이드

## 당신의 AI 팀이 일하고 있습니다

회사에 소속된 AI 에이전트(CEO, CMO, Content Marketer 등)가 24시간 업무를 수행합니다.
당신은 **텔레그램**으로 지시하고, **Paperclip 대시보드**에서 직접 관리할 수 있습니다.

---

## 1. 텔레그램 사용법

텔레그램에서 봇을 추가하면 바로 사용할 수 있습니다.

### 시작하기
1. 텔레그램에서 전달받은 봇 이름 검색
2. `/start` 입력 → 연결 완료

### 명령어

| 명령 | 설명 |
|---|---|
| `/iss` | 이슈(업무) 목록 보기 |
| `/iss SOC-XX` | 특정 이슈 상세 보기 |
| `/newiss` | 새 이슈(업무) 만들기 |
| `/inbox` | 알림 확인 (에이전트가 보고한 것) |
| `/heartbeat` | 에이전트 활동 시작 |

### 일반 대화
명령어 없이 메시지를 보내면 AI 비서가 답변합니다.

```
"컨텐츠 기획 프로젝트 상황 알려줘"
→ AI 비서가 현황을 요약해서 알려드립니다.

"SOC-5 이슈에 댓글 달아줘: 블로그 초안 확인했습니다"
→ 해당 이슈에 댓글이 등록됩니다.
```

### 승인 처리
에이전트가 승인을 요청하면 텔레그램에 버튼이 뜹니다.
- [승인] 버튼 → 승인
- 텍스트 입력 → 댓글로 피드백
- 무응답 → 스킵

---

## 2. Paperclip 대시보드

웹 브라우저에서 직접 접속하여 관리합니다.

### 접속 방법
- PC에서: 자동으로 열려 있는 브라우저 탭
- 모바일에서: Tailscale 앱 설치 후 안내받은 주소 접속

### 주요 메뉴

| 메뉴 | 설명 |
|---|---|
| **Dashboard** | 에이전트 현황, 최근 활동, 이슈 통계 |
| **Issues** | 업무(이슈) 목록 — 생성, 수정, 댓글, 상태 변경 |
| **Inbox** | 에이전트가 보고한 알림 |
| **Org** | 조직도 — 에이전트(AI 직원) 구성 확인 |
| **Settings** | 회사 설정 |

### 이슈 관리
1. **Issues** 클릭
2. **New Issue** → 제목 입력, 담당자 지정, 프로젝트 선택
3. 이슈 클릭 → 상세 내용 확인, 하단에 댓글 입력

### 에이전트에게 일 시키기
1. 이슈를 만들고 담당자를 지정하면 에이전트가 자동으로 작업 시작
2. 진행 상황은 이슈 댓글에 업데이트됨
3. 승인이 필요한 건은 텔레그램으로 알림

---

## 3. 자주 묻는 질문

**Q: 에이전트가 안 움직여요**
→ `/heartbeat` 명령으로 활동을 시작시킬 수 있습니다.

**Q: 이슈를 잘못 만들었어요**
→ Paperclip 대시보드에서 이슈를 열고 수정하거나 닫을 수 있습니다.

**Q: 텔레그램 응답이 느려요**
→ AI가 분석 중일 수 있습니다. 잠시 기다려주세요.

**Q: 모바일에서 대시보드가 안 열려요**
→ Tailscale 앱이 연결되어 있는지 확인해주세요.
