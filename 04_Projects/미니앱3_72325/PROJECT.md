# PROJECT: 미니앱3 - 텔레그램 기반 에이전트 원격 제어 시스템

## 1. 프로젝트 개요 (Overview)
본 프로젝트의 목표는 텔레그램 미니앱(Telegram Mini App)을 개발하여, 사용자가 원격으로 MyCrew 에이전트의 프로젝트 관리 및 태스크 제어를 수행할 수 있는 인터페이스를 구축하는 것입니다. 사용자는 이 미니앱을 통해 프로젝트를 선택하고, 칸반 보드 형식으로 태스크를 관리하며, 에이전트가 생성한 결과물을 확인, 승인 또는 재작업을 지시할 수 있습니다.

## 2. 목표 (Goals)
- **직관적인 원격 제어**: 텔레그램 내에서 에이전트 프로젝트 및 태스크를 직관적으로 관리할 수 있는 UI/UX 제공.
- **실시간 상호작용**: 태스크 상태 변경, 결과물 도착 등 주요 이벤트에 대한 실시간 알림 및 업데이트 기능 구현.
- **안정적인 데이터 연동**: MyCrew 백엔드 시스템과 텔레그램 미니앱 간의 안전하고 효율적인 API 연동 구축.
- **엄격한 데이터 격리**: `strict_isolation` 정책을 준수하여 프로젝트 데이터의 보안 및 격리 보장.

## 3. 주요 기능 범위 (Scope)
- 프로젝트 목록 조회 및 선택 기능
- 칸반 보드 기반 태스크 관리 (생성, 상태 변경, 상세 조회)
- 태스크 결과물 확인 및 피드백 (승인/재작업) 기능
- 텔레그램 Bot을 통한 알림 기능

## 4. 성공 지표 (Success Metrics)
- 사용자가 미니앱을 통해 월 100건 이상의 태스크를 성공적으로 처리.
- API 평균 응답 시간 500ms 미만 유지.
- QA 단계에서 발견되는 Critical Bug 0건.
