# 🗂 Image Lab — 소시안 레퍼런스 이미지 보관함

> **경로**: `06_소시안자료/lab_references/`
> **목적**: Image Lab의 "소시안 자산 라이브러리" 탭에서 자동으로 불러오는 기준 이미지 폴더

---

## 사용 방법

이 폴더에 이미지를 추가하면 MyCrew Image Lab의
**[🗂 소시안 자산 라이브러리]** 탭에 자동으로 표시됩니다.

```
Image Lab → 레퍼런스 선택 →
[📁 직접 업로드] 또는 [🗂 소시안 자산 라이브러리] ← 이 폴더
```

---

## 권장 파일 구조

```
lab_references/
├── 캐릭터/           ← 소시안 캐릭터, 픽셀아트 스프라이트
├── 광고소재/         ← 실제 집행된 배너, 카카오/네이버/구글
├── SNS콘텐츠/        ← 인스타 피드, 릴스 썸네일
└── 브랜드/           ← 로고, 워드마크, 브랜드 컬러 기준 이미지
```

---

## 지원 포맷

- PNG (투명배경 지원 — 캐릭터 스프라이트 권장)
- JPG / JPEG
- WebP
- 최대 파일 크기: 5MB

---

## 백엔드 연동 방식 (루카가 구현)

`GET /api/imagelab/assets` 엔드포인트가 이 폴더를 스캔하여
파일 목록과 썸네일 URL을 반환합니다.

```json
{
  "assets": [
    { "name": "소시안_캐릭터_01.png", "url": "http://localhost:4000/lab-assets/소시안_캐릭터_01.png", "category": "캐릭터" },
    { "name": "배너_네이버_01.jpg",   "url": "http://localhost:4000/lab-assets/배너_네이버_01.jpg",   "category": "광고소재" }
  ]
}
```

---

*이 폴더의 이미지는 LUMI 프롬프트 트레이닝의 기준 데이터셋이 됩니다.*
