# 소시안 인스타그램 릴스 워크플로우

## 🎯 목표
소시안 브랜드 릴스/쇼츠 스크립트를 Hook → 본론 → CTA 구조로 자동 생성한다.

---

## 📋 실행 단계

### Step 1: Hook 발굴 (PICO)
```
담당: PICO (콘텐츠 발산)
방식: 높은 창의성 모드 — 형식 제약 없이 쏟아내기
출력:
  - FOMO형 Hook 3개 (예: "아직도 DM 직접 보내요?")
  - 의문형 Hook 3개 (예: "팔로워 1만명인데 매출이 0원?")
  - 역설형 Hook 2개 (예: "팔로워 늘릴수록 손해인 이유")
```

### Step 2: Hook 선정 + 스크립트 작성 (NOVA)
```
담당: NOVA (마케팅 지능)
CCB 참조: PICO의 Hook 후보 + 최근 성공 패턴 (Self-Learning 로그 참조)
출력 (30초 기준):
  [0~3초]  Hook 문장 (선정된 1개)
  [3~10초] 공감 상황 묘사 (타겟의 Pain)
  [10~22초] 소시안 기능 시연 or 결과 수치
  [22~30초] CTA ("링크 클릭하면 무료 체험")
자막 텍스트: 전체 스크립트 자막 버전 별도 작성
```

### Step 3: 포맷 최적화 (LUMI)
```
담당: LUMI
체크리스트:
  □ 자막 텍스트 15자 이내 한 줄 단위 분절
  □ 핵심 숫자/키워드 강조 표시 (이모지 or 대문자)
  □ 해시태그 10개 이내 (인스타 변동 정책 반영)
  □ 섬네일 키워드 제안 (정방형 1080×1080 기준)
출력: 발행 준비 완료 스크립트 + 자막 파일
```

---

## 🏆 승자 레시피 (검증된 패턴)
- **FOMO + 의문형 조합** → 저장율 3배 (2026-04 실험 확인)
- **숫자 포함 Hook** → 1초 이탈율 감소 효과
- **CTA "무료 체험"** > "지금 바로" (클릭율 비교 우위)

## ⛔ 실패 패턴 (금지)
- 역발상형 Hook 단독 사용 → 소시안 브랜드 톤과 불일치
- 30초 초과 스크립트 → 알고리즘 패널티
- 경쟁사 직접 비교 언급 → 법적 리스크
