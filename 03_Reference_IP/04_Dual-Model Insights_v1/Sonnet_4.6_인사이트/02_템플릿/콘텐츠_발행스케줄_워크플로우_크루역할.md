# 📅 파트 1: 콘텐츠 발행 스케줄

## 전체 캠페인 타임라인 (4주 기준)

```
D-7  (이벤트 1주 전)   → 사전 예고 콘텐츠 발행
D-3  (이벤트 3일 전)   → 본격 홍보 시작
D-Day (이벤트 시작)    → 전 채널 동시 공지
D+7  (이벤트 중반)     → 리마인드 + 중간 결과 공유
D+14 (이벤트 마감)     → 마감 임박 알림 + 결과 발표
```

# 🔄 파트 2: 캠페인 워크플로우

## 워크플로우 1: 쿠폰 이벤트 (댓글 → 자동DM → 쿠폰 발송)

```
[준비 단계 — D-7 이전]
1. 쿠폰 코드 생성 및 소시안 시스템 등록
   → 담당: 개발팀 / 소시안 대시보드 담당자
   
2. 소시안 댓글DM 설정
   → 트리거 키워드: "쿠폰"
   → 자동 DM 내용: "참여 감사합니다! 쿠폰 코드는 {{코드}}입니다."
   → 선착순 캡 설정 (N명 초과 시 다른 메시지 발송)
   
3. 콘텐츠 제작 완료
   → 캐러셀 이미지 A / 릴스 영상 / 블로그 포스팅 초안 완성

[실행 단계 — D-Day]
4. 전 채널 동시 발행 (10:00 AM 기준)
   → 인스타 캐러셀 게시 + 댓글DM 활성화
   → 블로그 포스팅 발행
   → 홈페이지 공지 배너 게시
   
5. 실시간 모니터링 (D-Day ~ D+2)
   → 댓글 수 / DM 발송 수 / 쿠폰 소진율 체크
   → 이상 발생 시 즉시 알림

[마감 단계 — D+14]
6. 이벤트 종료 공지
   → 소시안 댓글DM 비활성화
   → 결과 발표 포스팅 발행
   → 쿠폰 미사용자 리마인드 DM 발송 (선택)
```

## 워크플로우 2: 블로그 리뷰 이벤트

```
[준비 단계]
1. 리뷰 제출 폼 생성 (Google Forms)
   → 필드: 블로그 URL, 이름, 연락처, 소시안 가입 이메일

2. 심사 기준 문서 작성
   → 글자 수 1,000자 이상
   → 소시안 화면 캡처 3장 이상
   → 필수 해시태그 포함 여부

[홍보 단계 — D-Day]
3. 홍보 채널 3종 동시 발행
   → 홈페이지: 공지 배너 + 링크 (배너: 홈페이지공지용-리뷰이벤트공지베너.png)
   → 블로그: 이벤트 공지 포스팅 (배너: 블로그대표이미지-리뷰이벤트베너8.png)
   → 인스타그램: 캐러셀 B 게시

[심사 단계 — 마감 후]
4. 제출물 수집 및 1차 기준 통과 여부 확인
5. 우수 리뷰 선정 (팀 내부 심사)
6. 당첨자 발표 + 혜택 지급

[결과 활용]
7. 우수 리뷰 공식 채널에 공유 (동의 받은 경우)
8. 리뷰 내용을 향후 마케팅 콘텐츠 소재로 활용
```

# 👥 파트 3: 크루별 역할과 참여 순서

## 캠페인 역할 정의

| 크루 역할 | MyCrew 에이전트 | 주요 업무 |
|---------|--------------|---------|
| **캠페인 기획자** | CMO (전략 에이전트) | 전체 일정 조율, KPI 설정, 의사결정 |
| **카피라이터** | 마케팅 에이전트 | 블로그 포스팅 초안, 캡션 작성 |
| **비주얼 디렉터** | 디자인 에이전트 | 캐러셀 이미지 기획, 릴스 구성안 |
| **영상 스크립터** | 영상 에이전트 | 릴스/쇼츠 스크립트 작성 |
| **SEO 전문가** | SEO 에이전트 | 블로그 키워드 최적화, 해시태그 |
| **데이터 분석가** | 분석 에이전트 | 성과 모니터링, 리포트 |

## 참여 순서 (순차 워크플로우)

```
[Phase 1: 전략 수립] ← CMO 에이전트
  ├── 캠페인 목표 설정 (KPI)
  ├── 이벤트 방식 확정 (쿠폰 이벤트 + 리뷰 이벤트)
  ├── 예산 및 쿠폰 수량 확정
  └── 전체 타임라인 승인

   ↓

[Phase 2: 콘텐츠 제작] ← 카피라이터 + 비주얼 디렉터 병렬 진행
  ├── (카피) 블로그 포스팅 3종 초안 작성
  ├── (카피) 인스타 캡션 + 캐러셀 텍스트 작성
  ├── (비주얼) 캐러셀 이미지 구성안 작성 → 디자이너 전달
  └── (영상) 릴스/쇼츠 스크립트 작성

   ↓

[Phase 3: SEO 최적화] ← SEO 에이전트
  ├── 블로그 제목/소제목 키워드 최적화
  ├── 해시태그 세트 확정 (채널별)
  └── 메타 디스크립션 작성

   ↓

[Phase 4: 검토 및 승인] ← CMO 에이전트 + 대표님
  ├── 콘텐츠 전체 리뷰
  ├── 오탈자/사실 관계 확인
  └── 최종 발행 승인

   ↓

[Phase 5: 발행 실행] ← 마케팅 에이전트 (발행 담당)
  ├── 채널별 스케줄에 맞춰 발행
  ├── 소시안 댓글DM 자동화 활성화
  └── 홈페이지 배너 게시 요청

   ↓

[Phase 6: 모니터링 & 최적화] ← 분석 에이전트
  ├── 일별 댓글 수 / DM 발송 수 / 쿠폰 소진율 추적
  ├── 블로그 조회수 / 유입 키워드 분석
  └── 성과 부진 채널 → 전략 수정 제안

   ↓

[Phase 7: 결과 보고] ← CMO 에이전트
  ├── 최종 성과 리포트 작성
  ├── 우수 리뷰 선정 및 시상
  └── 다음 캠페인 인사이트 도출
```

## KPI (성과 지표)

| 지표 | 목표값 | 측정 방법 |
|------|-------|---------|
| 댓글 수 (쿠폰이벤트) | {{N}}개 이상 | 인스타 인사이트 |
| DM 발송 수 | 댓글 수의 95% 이상 | 소시안 대시보드 |
| 블로그 리뷰 제출 수 | {{N}}건 이상 | 제출 폼 |
| 블로그 포스팅 조회수 | {{N}}회 이상 | 블로그 통계 |
| Plan C 전환율 | 쿠폰 수령자의 {{N}}% | 소시안 구독 데이터 |
