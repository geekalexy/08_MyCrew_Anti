```

나레이션: "소시안 이벤트 참여해봤는데 진짜 신기했어요"

[장면 2 — 5~25초: 화면 녹화 시연]
화면: 인스타 게시물 → 댓글 "쿠폰" 입력 →
      DM 수신 화면 (쿠폰 코드 보이게)
자막: "댓글 달고 30초도 안 됐는데 DM이 왔어요"
나레이션: "이게 소시안 댓글DM 자동화예요.
           직접 보내는 게 아니라 자동이에요"

[장면 3 — 25~45초: 이벤트 혜택 설명]
자막 카드:
  "지금 이벤트 중 🎁"
  선착순 → Plan C 1개월 무료
  전체 참여 → 할인 쿠폰
나레이션: "지금 소시안 인스타에서 이벤트 중인데
           쿠폰 댓글 달면 Plan C 체험권도 받을 수 있어요"

[장면 4 — 45~60초: CTA]
자막: "소시안 팔로우하고 '쿠폰' 댓글 달기"
화면: @socian_official 프로필 화면
나레이션: "링크는 설명란에 있어요!"

제목: 인스타 댓글 달면 DM 자동으로 온다고?? 소시안 이벤트 체험기
태그: #소시안 #DM자동화 #공구셀러 #인스타이벤트 #쿠폰이벤트
```
