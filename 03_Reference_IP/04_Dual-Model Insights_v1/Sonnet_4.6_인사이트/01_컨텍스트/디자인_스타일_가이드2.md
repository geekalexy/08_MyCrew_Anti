## 1. 컬러 시스템 (Color System)

### 주요 색상 (Primary)
| 역할 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| Primary Blue | 소시안 블루 | `#3B7BF5` | CTA버튼, 강조 텍스트, 핵심 UI |
| Background | 화이트 | `#FFFFFF` | 페이지 배경 |
| Dark Navy | 다크 네이비 | `#0F1F4B` | 푸터, 섹션 배경 |

### 보조 색상 (Secondary)
| 역할 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| Light Gray BG | 연회색 | `#F5F7FA` | 섹션 구분 배경 |
| Text Dark | 다크 그레이 | `#1A1A2E` | 본문 텍스트 |
| Text Medium | 미디엄 그레이 | `#6B7280` | 서브 텍스트, 설명 |
| Border | 라이트 그레이 | `#E5E7EB` | 카드 보더, 구분선 |

### 컬러 원칙
- 배경은 **화이트 또는 연회색** 위주 → 콘텐츠 가독성 최우선
- CTA 요소에는 **소시안 블루** 단일 사용으로 집중도 유지
- 푸터/강조 섹션에는 **다크 네이비**로 신뢰감 형성

## 2. 타이포그래피 (Typography)

### 폰트 패밀리
- **한국어**: Noto Sans KR (또는 Pretendard)
- **영문/숫자**: Inter, Noto Sans

### 텍스트 스케일
| 레벨 | 크기 | 굵기 | 용도 |
|------|------|------|------|
| Hero Title | 40–48px | 700 (Bold) | 히어로 섹션 메인 헤드라인 |
| H2 | 28–32px | 700 | 섹션 제목 |
| H3 | 20–24px | 600 | 서브 섹션, 카드 제목 |
| Body | 15–16px | 400 | 본문 텍스트 |
| Caption | 13–14px | 400 | 보조 설명, 레이블 |

- **최대 너비**: 1200px

- **모바일 분기점**: 768px
- **섹션 수직 패딩**: 80px ~ 120px
- **카드 내부 패딩**: 24px ~ 32px
- **간격 단위**: 8px 기반 (16 / 24 / 40px)

## 4. 컴포넌트 스타일

### 버튼
```
Primary CTA:
  Background: #3B7BF5 | Text: #FFFFFF
  Border Radius: 8px | Padding: 12px 28px | Font: 16px/600

Secondary:
  Background: transparent | Border: 1px solid #3B7BF5
  Text: #3B7BF5
```

### 카드
```
Background: #FFFFFF | Border: 1px solid #E5E7EB
Border Radius: 12px | Shadow: 0 2px 12px rgba(0,0,0,0.06)
Padding: 24–32px
```

## 5. SNS/블로그 콘텐츠 스타일 (블로그 분석 신규 추가)

### 네이버 블로그 포스팅 스타일
- **제목 구조**: `[핵심 기능/이슈] + 부연설명 + 이모지`
  - 예: `클릭율 높이는 신기능 "리타겟팅DM" 출시 🎯`
  - 예: `소시안 DM 분석 서비스 오픈!`
- **본문 구조**: 공지 → 혜택 설명 → 참여 방법 → 유의사항 → CTA
- **이모지 사용 규칙**:
  - 섹션 구분에 이모지 아이콘 활용 (📅 🎁 ✍️ 📌)
  - 과도하지 않게, 가독성 우선
- **강조 표현**: 핵심 수치/혜택은 굵게 또는 따로 단락 구분

### 인스타그램 콘텐츠 스타일
- 카드뉴스 형식 적합 (블루+화이트 배경)
- 핵심 메시지 1개 → 짧고 임팩트 있게
- 필수 해시태그 포함 (브랜드_가이드라인2.md 참조)

## 6. 이미지 & 모션

### 이미지 스타일
- 실제 서비스 화면 스크린샷 적극 활용 → 신뢰도 강화
- 디바이스 목업 (스마트폰 프레임) 사용
- 배경: 연회색(`#F5F7FA`) 또는 화이트
- 기하학 도트 패턴으로 기술적 느낌 강조

### 모션
- Ease In-Out, 200–300ms
- 호버 효과: 버튼 Darken 10%, 카드 Shadow 강화
