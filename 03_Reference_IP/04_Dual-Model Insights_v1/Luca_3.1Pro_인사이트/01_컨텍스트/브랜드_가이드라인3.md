# 🏷️ 브랜드 가이드라인 (Brand Guidelines)

## 1. 브랜드 미션
"누구나 장벽 없이 가장 유연하고 강력한 인스타 자동화(다중 키워드, 리타겟팅, 이벤트 추첨)를 도입하여, 매출을 올리고 팬덤을 관리할 수 있게 한다."

## 2. 브랜드 페르소나 (Tone & Manner)
- **친절한 가이드 (Friendly Guide)**: GitBook 매뉴얼과 블로그에서 보여지듯, 처음 자동화를 접하는 유저라도 시크릿 모드 설정부터 인스타그램 권한 연동까지 막힘없이 할 수 있도록 단계별로 세심하게 챙겨주는 친절한 도우미 톤.
- **합리적이고 당당한 (Reasonable)**: "무료 체험부터 써보세요"라는 워딩에서 나오는 제품력에 대한 자신감. 타겟의 사업 규모에 맞춘 프로 플랜 업셀링의 자연스러움.
- **전문가적 확장 (Professional)**: 단순 인사DM이 아니라 '리타겟팅', '다중 키워드 분류' 등 고도화된 마케팅 기법을 쉽게 풀어내는 마케터 보이스.

## 3. 핵심 전략 키워드
- 소시안 (Socian)
- 무료 체험 (Free Trial) & 프로 플랜
- 매니챗보다 쉬운 자동디엠 
- 다중 키워드 / 리타겟팅 DM / 이벤트 추첨
- 인스타 릴스 자동 업로드

## 4. 커뮤니케이션 방향성
- **도입 장벽 허물기**: 요금제 페이지의 무료 체험(Free Trial) 소구점을 적극 활용하여 최초 전환을 유도. 
- **기능의 실효성 증명**: 깃북(Gitbook)에 나열된 세부 기능들이 실제 공구 셀러의 하루 일과표를 어떻게 단축해주는지 블로그를 통해 케이스 스터디로 증명.
