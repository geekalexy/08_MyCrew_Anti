# 🚀 MyCrew 에이전트 사용자 가이드 (Commercial Version)
**귀사의 팀에 '경력직 AI 크루'가 합류한 것을 환영합니다!**

MyCrew는 단순한 소프트웨어가 아닙니다. 수천 번의 실전 업무를 통해 단련된 문제 해결 지혜를 가진 디지털 인재들의 모임입니다. 본 가이드는 귀사에 배포된 MyCrew 에이전트를 성공적으로 구동하고 소통하는 방법을 안내합니다.

---

## 🏗️ 1. 환경 설정 (Setup)

### 1.1 필수 요구 사항
* **Node.js**: v18.0.0 이상 (권장 v20+)
* **공간**: 최소 1GB 이상의 디스크 여유 공간
* **네트워크**: 외부 API(Telegram, Google Gemini 등) 연결 가능 환경

### 1.2 설치 및 구동
1. 전달받은 패키지 파일을 해제합니다.
2. 터미널(Terminal)에서 해당 폴더로 이동합니다.
3. 의존성 패키지를 설치합니다:
   ```bash
   npm install
   ```
4. 에이전트 서버를 구동합니다:
   ```bash
   node server.js
   ```

## 🧠 2. 자아(Soul) 설정: MYCREW.md
MyCrew 크루들의 행동 강령을 정의하는 핵심 파일입니다. `01_Company_Operations/04_HR_온보딩/MYCREW.md` 파일을 수정하여 귀사의 비전을 입력하세요.
* **Mission**: 우리 회사가 달성하려는 궁극적인 목표
* **Core Values**: 업무를 처리할 때 반드시 지켜야 할 가치
* **Rules**: 금기 사항 및 반드시 준수해야 할 프로세스

## 💬 3. 대화 및 업무 지시
귀사 전용 **텔레그램 봇**을 통해 실시간으로 업무를 지시할 수 있습니다.
* **직군별 자동 라우팅**: 마케팅, 분석, 디자인 등 메시지를 분석하여 최적의 크루가 응답합니다.
* **성장 확인**: 대화 하단에 달리는 `Score`는 에이전트가 얼마나 만족스럽게 임무를 수행했는지 나타냅니다. 고득점 작업은 에이전트의 스킬 라이브러리에 자동으로 기록되어 다음번에 더 나은 결과를 보여줍니다.

## 🔒 4. 보안 및 데이터 정책
* **데이터 격리**: 귀사의 데이터는 인가되지 않은 외부로 절대 유출되지 않습니다.
* **비식별 학습**: 에이전트가 자가 학습을 수행할 때, 고유명사와 숫자는 자동으로 익명화 처리되어 '지능(패턴)'만 남고 '기록(로그)'은 파기됩니다.
