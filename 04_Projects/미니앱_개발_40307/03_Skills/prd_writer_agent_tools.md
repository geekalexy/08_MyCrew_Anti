# PRD Writing for Agent Tools

비즈니스 목표(에이전트 원격 제어)를 개발 가능한 단위의 명확한 기능 명세(User Stories, Acceptance Criteria)로 변환하는 PRD(Product Requirements Document)를 작성한다. 다른 AI 에이전트들이 오해 없이 해석하고 즉시 개발에 착수할 수 있도록 구체적이고 정형화된 형식으로 기술한다.
