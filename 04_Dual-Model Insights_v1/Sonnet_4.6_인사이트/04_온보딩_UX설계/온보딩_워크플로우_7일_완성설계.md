## Day 2 — 템플릿 자료 요청 시작

### [트리거] 컨텍스트 완료 후 다음날 오전 9시 or 재방문 시

---

**① 아리 채팅 — 중요성 각인 + 전환**

```
아리:
컨텍스트 세팅 잘 되셨죠? 😊

이제 두 번째이자 마지막 단계예요.

솔직하게 말씀드리면 —
크루 결과물 만족도의 90%가
컨텍스트 + 기존 콘텐츠 자료에서 나와요.

컨텍스트는 어제 완료됐고,
오늘 기존 콘텐츠 자료만 주시면
크루가 진짜 소시안처럼 쓰기 시작해요.

AI가 아무리 똑똑해도
기존에 소시안이 써온 말투, 이미지 스타일,
실제 쓴 표현들을 봐야 "소시안 글"이 나오거든요.

딱 5분만요. 준비되셨으면 시작해요! 🚀
```

---

**② 자료 수집 채팅 3단계**

```
[단계 1] 기존 콘텐츠 채널 파악
아리:
먼저 — 소시안은 어떤 채널에 콘텐츠를 올리고 계세요?

  ☐  네이버 블로그
  ☐  인스타그램 (피드/릴스)
  ☐  유튜브
  ☐  티스토리
  ☐  기타 →

(없는 것은 체크 안 하셔도 돼요)
```

```
[단계 2] 기존 파일 요청
아리:
좋아요! 그럼 혹시 기존에 만드신
블로그 글이나 인스타 이미지 파일이 있으면 주세요.

파일이 많으면 폴더 경로만 알려주셔도 돼요.
없으면 URL만으로 진행할게요!

→  파일 첨부 or 폴더 경로
→  "없어요" / "URL로 해주세요"
```

```
[단계 3] 접수 완료
아리:
받았어요! 📦

자료 제출 카드에서 한 번 더 확인해주시고
"제출" 눌러주시면 크루가 바로 분석 시작할게요.

[📋 자료 제출 카드 열기 →]
```

## Day 3 — 자료 분석 완료 + 첫 결과물 미리보기

### [트리거] 자료 제출 후 크루 분석 완료 시

```
아리:
🎊 자료 분석 완료!

크루가 소시안 스타일을 완전히 파악했어요.

분석한 내용을 간단히 공유드릴게요:

  ✅ 블로그 말투: 정보형 + 이모지 구분선 활용
  ✅ 캐러셀 구성: 4~5장, 블루/화이트 배경
  ✅ 릴스 구성: 후킹(3초) → 설명 → CTA 패턴
  ✅ 핵심 CTA: 카카오톡 문의, 무료체험 유도

이제 진짜 소시안스러운 콘텐츠를 만들 준비가 됐어요!

첫 번째 템플릿을 바로 드릴게요 👇
[📋 블로그 템플릿 v1 확인하기]
```

📅 2주마다 자료 업데이트 알림이 와요.
   새 기능, 이벤트, 캠페인이 생기면 알려주세요.
   크루가 항상 최신 소시안으로 일해요.

# 🗺️ 전체 흐름 요약 다이어그램

```
DAY 0 ──────────────────────────────────────────────────
  [온보딩 위자드]
    → 웰컴 오버레이
    → 아리 채팅 시작 (핵심고객 + 문제 수집)
    → URL 수집 → 웹/블로그 자동 파싱
    → "컨텍스트 세팅 중" 카드 생성
    → 분석 완료 알림

DAY 1 ──────────────────────────────────────────────────
  [SECTION 1 완료]
    → 컨텍스트 파일 3종 자동 생성
    → 크루에게 컨텍스트 지식 탑재
    → Section 2 시작 안내 채팅 발송

DAY 2 ──────────────────────────────────────────────────
  [SECTION 2 시작]
    → 아리 채팅 (90% 중요성 각인)
    → 기존 채널 파악 → 파일 요청
    → "기존 콘텐츠 자료 제출" 카드 생성

DAY 3 ──────────────────────────────────────────────────
  [자료 분석 + 1차 결과]
    → 크루가 기존 자료 분석 완료
    → 분석 결과 채팅으로 공유
    → 첫 번째 템플릿 생성 완료
    → 결과물 확인 카드 생성

DAY 4~5 ────────────────────────────────────────────────
  [결과물 확인 + 피드백]
    → 사용자 확인 → 피드백 입력
    → 크루 수정 → 최종 템플릿 확정

DAY 7 ──────────────────────────────────────────────────
  [SECTION 2 완료 → 정식 운영]
    → 온보딩 완료 메시지
    → 업데이트 주기 알림 활성화
    → 대시보드 정상 모드 전환
```
